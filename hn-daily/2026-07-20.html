The 10 highest-rated articles on Hacker News on July 20, 2026 which have not appeared on any previous Hacker News Daily are listed below, with Chinese summaries for quick reading.

  1. Minecraft: Java Edition now uses SDL3
    comments

    Minecraft 26.3 Snapshot 4 将底层窗口管理、输入与平台集成库从 GLFW 切换为 SDL3，在 Linux 系统上优先原生支持 Wayland，并默认采用无边框全屏模式。本次更新新增了自定义熔炉及炼药燃料、告示牌文本、村民食物与实体可见性等多项数据组件，数据包和资源包版本分别升级至 111.0 与 92.0。此外，调试叠加层新增了玩家移动速度显示并支持独立 GUI 缩放，旁观者模式玩家现可与传送门交互进行传送。

  2. Claude Fable produced a counterexample to the Jacobian Conjecture
    comments

    推特用户 levent 宣称其朋友 Claude Fable 构造出了雅可比猜想（Jacobian Conjecture）的反例，证明该猜想并不成立。该反例提供了一个从三维复数空间映射到自身的具体多项式映射，其雅可比行列式恒等于 -2。具体而言，该映射会将三个不同的三维坐标点同时映射至同一个目标点 (-1/4, 0, 0)，从而打破了猜想所需的单射性。

  3. Xiaomi-Robotics-1
    comments

    Xiaomi-Robotics-1 结合超过 10 万小时的非具身（UMI）数据预训练与少量真实机器人后训练，有效突破了机器人策略模型的训练数据瓶颈。模型在预训练阶段利用视觉语言模型自动标注管线学习通用动作生成，后训练阶段通过跨具身数据完成具身与指令对齐，且预训练扩展效应可直接且可预测地提升真实机器人的表现。在下游评估中，该模型仅需平均不到 10 小时的真实示范即可实现 75% 的新任务成功率，并在 RoboCasa、VLABench 等四个主流仿真基准测试中均达到 SOTA 表现。

  4. Exploit brokers pay $500k for WordPress RCEs. I found one with GPT5.6 and $25
    comments

    Searchlight Cyber 的研究员利用调整后的 LLM 提示词，以约 25 美元成本在 WordPress 中发现了一个未授权远程代码执行（RCE）漏洞链。漏洞起源于 WordPress 批处理 API（Batch API）在处理异常请求时导致验证数组与执行数组的索引错位，从而绕过了参数清理机制。攻击者进而结合 REST API 中 author__not_in 参数对标量输入缺乏转义的缺陷引发 SQL 注入，最终提权实现 RCE。

  5. Airport Simulator
    comments

    Airport Simulator 是一款由开发者 @lapunen 推出的网页端极简空中交通管制模拟游戏，灵感源自经典游戏 Flight Control。玩家需要通过拖曳绘制航线，引导不同颜色的飞机安全降落至对应颜色的跑道，同时管理航班起飞并防范空中碰撞事故。随着游戏进程推进，空中交通密度与起降节奏会迅速加快，高度考验玩家的实时调度反应与极限持久力。

  6. The EU is about to sell our most sensitive data to the US for visa-free travel
    comments

    欧盟委员会正与美国协商“增强型边境安全伙伴关系”（EBSP）框架协议，拟通过自动共享生物识别数据库等敏感数据，以维持欧洲公民赴美免签资格。泄露的草案显示欧盟大幅妥协，几乎完全满足了美方无限制调阅信息的要求，严重违反欧盟《基本权利宪章》及相关法律。此举可能导致旅行者遭受严密监控与政治审查画像，欧洲数字权利组织（EDRi）为此呼吁欧盟坚决抵制美方施压并保护公民隐私。

  7. Annoying and alarming things about OpenCode
    comments

    文章深入剖析了热门开源 AI 编程 Agent 工具 OpenCode 在设计与使用体验上的诸多缺陷，强烈建议用户停止使用该工具。作者指出 OpenCode 在 Prompt Cache 缓存管理上非常低下，频繁重新读取配置文件、粗糙的上下文剪枝与压缩机制会导致极高的重算延迟与资源浪费。此外，其在系统提示词、缺乏“拒绝”选项的权限确认机制、子 Agent 交互及工具 RPC 的设计上也存在大量体验极差与逻辑混乱的问题。

  8. Hacker wipes Romania's land registry database
    comments

    黑客 ByteToBreach（真实身份被曝光为阿尔及利亚人 Zakaria Mahdjoub）在对罗马尼亚国家地籍与房地产广告局（ANCPI）勒索失败后，利用有效凭证入侵并擦除了该国的整个地籍数据库及系统备份。该事件导致罗马尼亚房地产市场瘫痪一周，公证员与民众无法记录交易或获取产权证明。目前官方正利用未受影响的离线备份重建整个网络，而黑客已将盗取的员工凭证和内部文档在黑客论坛上公开出售。

    comments

  10. Kimi Work
    comments
