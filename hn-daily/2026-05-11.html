Daily Hacker News for 2026-05-11

The 10 highest-rated articles on Hacker News on May 11, 2026 which have not appeared on any previous Hacker News Daily are listed below, with Chinese summaries for quick reading.

  1. Driver accused of DUI tracks missing laptop to Illinois State trooper's house
    comments

    餐厅主管 Sherard Holland 在被曾获“顶级警察”称号的伊利诺伊州警 Kevin Bradley 逮捕后，通过定位系统在该警官家中找回了失踪的 MacBook，且其随后的酒驾指控被法院判决不成立。调查显示 Bradley 办理的 DUI 案件中有过半数被撤诉或判无罪，他曾利用巨额出庭加班费使年薪翻三倍至 25 万美元，目前正因涉嫌造假和侵犯民权面临起诉。

  2. Incident Report: CVE-2024-YIKES
    comments

    CVE-2024-YIKES 事故报告详细记录了一场涉及 JavaScript、Rust 和 Python 生态系统的供应链攻击，起因于维护者丢失物理 2FA 密钥及遭遇钓鱼。攻击通过深层嵌套依赖传播并感染了约 400 万名开发者，最终却被一个无关的加密货币挖矿蠕虫因强制升级依赖而意外“修复”。该讽刺性报告深入剖析了现代软件开发中凭证保护不足、小微库泛滥以及自动化合并等系统性安全隐患。

  3. Obsidian plugin was abused to deploy a remote access trojan
    comments

    安全研究人员发现一起针对金融和加密行业的定向社交工程攻击（REF6598），攻击者通过社交平台冒充风险投资人，诱导受害者协作使用恶意的 Obsidian 共享库。一旦受害者开启插件同步功能，便会通过被篡改的社区插件在系统中植入 PHANTOMPULSE 远程访问木马。该木马具备截屏和文件窃取能力，且利用以太坊区块链动态解析其 C2 服务器地址，使其指令接收机制具有极强的去中心化和抗干扰特性。

  4. Running local models on an M4 with 24GB memory
    comments

    Johanna Larsson 分享了在 24GB 内存的 M4 MacBook 上利用 LM Studio 运行本地大模型的实践，指出 Qwen 3.5-9B 量化版是目前综合表现最佳的模型，可实现每秒 40 token 的生成速度及 128K 上下文。通过配置思考模式和特定推理参数，该设置能有效处理 Elixir 代码修复等任务，但在应对 Git 冲突解决等复杂逻辑时仍需人工干预。作者强调本地模型虽然在独立解决问题上不及 SOTA 模型，但在无网办公、隐私保护和零订阅成本方面具有独特吸引力。

  5. I'm going back to writing code by hand
    comments

    作者复盘了使用 Claude 历时七个月开发 Kubernetes TUI 工具 k10s 的经历，因项目最终演变为逻辑混乱、难以维护的“上帝对象”而被迫从零重写。他指出 AI 擅长快速堆砌功能而非构建系统架构，若无人类预设的接口和规则约束，代码库会因状态泄露和特例分支在复杂度增长时迅速崩溃。作者建议开发者应通过指令文件为 AI 设定明确的架构规约和所有权守则，由人类主导顶层设计以确保 AI 辅助开发的可持续性。

  6. Mythos Finds a Curl Vulnerability
    comments

    Anthropic 声称其新 AI 模型 Mythos 具有极强的安全漏洞发现能力，curl 首席开发者 Daniel Stenberg 受邀利用该模型对 curl 代码库进行了深度扫描。尽管 Mythos 报告了 5 个“确认”漏洞，但 curl 团队核实后仅认同一项低风险 CVE，并认为其表现并未显著优于此前已使用的 AISLE 或 Codex 等 AI 工具。Daniel 总结指出，虽然 Mythos 的危险性存在营销夸大嫌疑，但 AI 驱动的分析器在识别已知类型的代码缺陷方面确实已远超传统静态分析工具。

  7. Gmail registration now requires scanning a QR code and sending a text message
    comments

    Google 账号注册流程近期发生重大调整，要求用户通过扫描二维码触发手机主动向官方发送验证短信，而非传统的接收验证码模式。这一改变旨在提升安全性并防范网络钓鱼攻击，但也直接导致 SMSpool 等第三方虚拟短信验证服务失效。隐私保护社区正深入探讨此举对匿名注册的影响，以及在全球不同电信实名制背景下，如何规避账号与真实身份的强关联风险。

  8. Ratty – A terminal emulator with inline 3D graphics
    comments

    Ratty 是一款基于 Rust 开发并利用 WGPU 进行 GPU 渲染的终端模拟器，其最大特色是支持行内 3D 图形显示和标志性的旋转老鼠光标。该项目深受 TempleOS 启发，通过 Ratatui 库构建，支持自定义着色器，旨在为现代终端体验引入高性能的 3D 演示模式。

  9. CUDA-oxide: Nvidia's official Rust to CUDA compiler
    comments

    Nvidia 推出的 cuda-oxide 是一个实验性的 Rust 转 CUDA 编译器，通过自定义 rustc 后端将标准 Rust 代码直接编译为 PTX，无需 DSL 或外部语言绑定。它允许开发者利用 Rust 的类型系统和所有权模型编写地道的 GPU 算子，并强调安全性。该工具还支持异步执行模式，可将 GPU 任务编排为延迟加载的任务图，并允许通过 .await 语法进行调度和结果等待。

  10. TanStack NPM Packages Compromised
    comments

    TanStack 旗下的多个 NPM 软件包（如 history、react-router 等）遭受大规模供应链攻击，恶意代码在安装时通过 prepare 脚本静默执行。该恶意程序会窃取 AWS、GCP、GitHub 凭据及 SSH 密钥，利用 Session/Oxen 网络进行加密外传，并能自动重发受感染包以实现自我传播。初步分析显示攻击者可能利用了 GitHub Actions 的 OIDC 信任发布机制，建议开发者核查 package.json 中是否存在指向特定 git 提交的恶意依赖指纹。
