The 10 highest-rated articles on Hacker News on April 22, 2026 which have not appeared on any previous Hacker News Daily are listed below, with Chinese summaries for quick reading.

  1. Acetaminophen vs. ibuprofen
    comments

    对乙酰氨基酚虽然治疗窗口较窄且过量服用易导致肝衰竭，但在按剂量使用时，其安全性通常优于布洛芬等非甾体抗炎药（NSAIDs）。布洛芬会全身性抑制COX酶，从而增加胃溃疡、心血管血栓以及在脱水或压力状态下发生肾损伤的风险，而对乙酰氨基酚对这些器官的影响极小。即使是肝病患者，在严格限速的情况下，医生也通常建议优先选择对乙酰氨基酚而非布洛芬，以规避潜在的多系统器官风险。

  2. Changes to GitHub Copilot individual plans
    comments

    GitHub 宣布暂停 Copilot Pro、Pro+ 及学生方案的新用户注册，并收紧现有个人方案的会话与周令牌（Token）使用限制。此次调整主因是“代理式工作流”导致计算资源消耗远超原计划成本，Pro+ 方案的限额现已提升至 Pro 的 5 倍以上。此外，Opus 模型将从 Pro 方案中移除，VS Code 和 CLI 已上线限额实时提醒，不接受变更的用户可在 5 月 20 日前申请按比例退款。

  3. Making RAM at Home [video]
    comments

    YouTube 频道 Dr.Semiconductor 在这段约 18 分钟的视频中，展示了在家庭环境下制造随机存取存储器（RAM）的全过程。视频详细介绍了制造技术，并参考了 Projects in Flight 关于半导体掺杂（doping）的相关资料。该内容自 2026 年 4 月发布以来已获得超过 52 万次播放，通过硬核实操展示了个人制造半导体器件的可能性。

  4. Meta to start capturing employee mouse movements, keystrokes for AI training
    comments

    Meta公司启动了一项名为“代理转型加速器”（ATA）的内部计划，通过捕捉员工的鼠标移动、点击和键盘按键等数据来训练AI模型，旨在提升智能体自主执行复杂办公任务的能力。该计划目前主要针对美国员工，采集范围还包括屏幕快照以提供操作上下文，帮助模型学习用户界面的导航与快捷键使用。尽管Meta承诺相关数据仅用于模型训练且不涉及绩效考核，但此举仍因隐私保护担忧及缺乏退出机制在公司内部引发了广泛争议。

  5. Claude Code to be removed from Anthropic's Pro plan?
    comments

    Anthropic 疑似已在其 20 美元/月的 Pro 订阅定价页面中移除 Claude Code 相关描述，引发开发者对该 AI 编程工具权限变更的担忧。目前该变动表现并不一致，部分现有订阅者和年付用户反馈仍可正常通过 CLI 访问，而新用户或未登录状态下则不再显示该功能。社区猜测这可能是 Anthropic 正在调整产品层级，计划将此高价值代理工具转为更高阶套餐或仅限 API 使用的前奏。

  6. SpaceX says it has agreement to acquire Cursor for $60B
    comments

    SpaceXAI 与 Cursor 达成深度合作，计划将 Cursor 的编程 AI 产品与 SpaceX 拥有百万块 H100 等效算力的 Colossus 超级计算机相结合，打造全球领先的编程与知识工作 AI 模型。根据协议，SpaceX 获得了在今年晚些时候以 600 亿美元收购 Cursor 的权利，或者选择为双方的合作支付 100 亿美元费用。

  7. Windows 9x Subsystem for Linux
    comments

    WSL9x 是一项允许现代 Linux 内核在 Ring 0 层与 Windows 9x 内核协作运行的技术，让老旧的 486 电脑无需硬件虚拟化即可并排运行 Windows 和 Linux 应用。该项目在技术思路上被类比为早期的 CoLinux 或 WSL1，即通过在内核层实现 POSIX 接口而非使用虚拟机。这种极客手段在复古硬件上实现了极高的集成度，被作者视为其最重大的黑客成就之一。

  8. GitHub CLI now collects pseudoanonymous telemetry
    comments

    GitHub CLI (gh) 现已开始收集伪匿名遥测数据，以获取功能使用情况的可见性，从而优化产品优先级并改进 AI 代理等新特性的设计。收集的数据涵盖系统架构、命令参数、随机生成的设备 ID 及执行环境信息，用户可通过设置环境变量查看具体的 JSON 发送载荷。若需关闭此功能，用户可将 GH_TELEMETRY 设为 false、启用 DO_NOT_TRACK 约定或通过 CLI 配置禁用，但需注意第三方扩展可能拥有独立的遥测机制。

  9. Qwen3.6-27B: Flagship-Level Coding in a 27B Dense Model
    comments

    通义千问团队正式开源 Qwen3.6-27B 稠密多模态模型，在 270 亿参数规模下实现了旗舰级的编程性能，在 SWE-bench 等核心基准测试中全面超越了前代 397B 参数的 MoE 旗舰模型。该模型原生支持图像与视频处理，具备强大的多模态推理与文档理解能力，且稠密架构显著降低了部署复杂度。目前该模型已提供开源权重并上线 API，可无缝集成至 OpenClaw 和 Claude Code 等主流 AI 编程助手。

  10. Alberta startup sells no-tech tractors for half price
    comments

    加拿大初创公司 Ursa Ag 推出售价仅为传统大牌一半的“无科技”拖放机，采用 90 年代翻新的康明斯机械发动机，完全剥离了电子元件和专有软件。该设计针对现代农机维修难、成本高的痛点，允许农民使用通用零件自行修理，有效解决了 John Deere 等品牌引发的“维修权”争议。尽管目前产能和分销网络尚在起步阶段，但其凭借 150 至 260 马力的高性价比机型，在北美农业市场引发了强烈关注。
