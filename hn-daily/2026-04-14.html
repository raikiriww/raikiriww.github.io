  1. US appeals court declares 158-year-old home distilling ban unconstitutional
    comments

    美国联邦第五巡回上诉法院裁定，有着158年历史的联邦家庭蒸馏禁令违宪，认为该禁令并非国会行使征税权的必要或适当手段。法院支持了业余蒸馏者协会（Hobby Distillers Association）的诉求，指出政府不能以防止逃税为由过度扩张联邦权力，甚至以此逻辑将各种居家活动刑事化。此裁决维持了德克萨斯州地方法院此前的判决，被视为个人自由和限制联邦权力边界的重要胜利。

  2. Make tmux pretty and usable (2024)
    comments

    本文详细介绍了通过编辑 ~/.tmux.conf 配置文件来增强 tmux 操作便捷性与视觉美感的基础方法。核心技巧包括将前缀键重映射为 Ctrl-a、自定义直观的窗格拆分快捷键（如 | 和 -），以及启用鼠标模式以支持快捷切换和窗格调整。此外，文章还提供了关于状态栏、边框及颜色样式的具体配置示例，帮助用户打造高效且个性化的终端环境。

  3. Nothing Ever Happens: Polymarket bot that always buys No on non-sports markets
    comments

    Nothing Ever Happens 是一个基于 Python 异步架构的 Polymarket 预测市场自动交易机器人，其核心策略是在非体育类独立市场中批量购买“No”（否定）选项。该工具集成了交易执行、实时仪表盘和自动状态恢复功能，并提供了详尽的脚本用于监控数据库统计、钱包历史及 Heroku 日志解析。项目内置了严格的安全校验模型，通过多重环境变量强制区分模拟与实盘模式，以确保链上私钥安全及交易流程的可控性。

  4. GitHub Stacked PRs
    comments

    GitHub 推出原生支持的 Stacked PRs 功能，允许将大型变更拆分为相互链接且独立审查的小型 PR 序列，以提升评审质量和合并速度。开发者可使用配套的 gh stack CLI 实现分支管理、级联变基及一键合并，同时 GitHub UI 提供了堆栈地图导航并强化了端到端 CI 验证。该工具链还支持 AI 代理集成，并能在合并部分 PR 后自动调整剩余堆栈的基准分支。

  5. DaVinci Resolve – Photo
    comments

    DaVinci Resolve 推出的全新照片页面将好莱坞顶级的调色工具引入静态摄影领域，支持 32K 源码处理及主流相机的原生 RAW 格式。该页面整合了魔术遮罩、深度图及 SuperScale 等先进 AI 技术，可实现 4 倍无损放大、后期灯光重绘与精细的人脸修饰。软件还支持相机联机拍摄与云端实时协作，并针对各平台 GPU 加速及硬件调色面板进行了深度优化，大幅提升了照片后期的处理效率。

  6. A new spam policy for “back button hijacking”
    comments

    Google 搜索团队宣布将“返回按钮劫持”（back button hijacking）纳入垃圾内容政策的“恶意行为”范畴，旨在打击干扰浏览器导航、阻止用户返回原页面的欺骗性操作。此类违规行为会导致站点面临人工处置或算法降权，从而影响其在搜索结果中的表现。该政策设有两个月的缓冲期，将于2026年6月15日正式生效，网站所有者需及时清理相关脚本及第三方广告插件。

  7. Backblaze has stopped backing up OneDrive and Dropbox folders and maybe others
    comments

    Backblaze 悄然更改了备份策略，停止支持 OneDrive、Dropbox、Google Drive 等云存储文件夹以及 .git 目录的自动备份，且未对受影响用户进行明确通知。尽管官方声称此举旨在优化性能，但这违背了其“备份所有文件且无类型限制”的长期承诺。作者认为这种隐瞒核心变更的行为严重背弃了用户信任，导致备份服务的可靠性大幅下降。

  8. jj – the CLI for Jujutsu
    comments

    jj 是 Jujutsu 分布式版本控制系统的命令行界面，它融合了 Git 与 Mercurial 的优点，旨在提供比 Git 更简单、易用且功能强大的操作体验。该工具通过更精简的架构支持更先进的版本管理功能，并拥有完全兼容 Git 的后端，允许个人用户在不影响团队其他人的情况下无缝切换使用。

  9. Rare concert recordings are landing on the Internet Archive
    comments

    芝加哥乐迷 Aadam Jacobs 将其自 20 世纪 80 年代以来录制的 1 万多盘珍贵现场演唱会磁带交给互联网档案馆进行数字化保存。目前已有约 2500 份录音上线，涵盖了 Nirvana、Sonic Youth 和 R.E.M. 等艺人早期罕见的演出片段。尽管原始录音设备简陋，但在志愿者团队的专业音频修复和标注下，这些磁带已转化为高质量的数字档案供公众收听。

  10. I wrote to Flock's privacy contact to opt out of their domestic spying program
    comments

    一名加州居民根据《加州消费者隐私法》(CCPA) 向车牌识别技术公司 Flock Safety 提出申请，要求其删除并停止采集涉及个人、车辆及家庭成员的所有数据。Flock Safety 以其仅作为“服务提供商”而非“数据控制者”为由拒绝了该请求，称数据归属于其客户，建议申请者直接联系相关组织。作者认为该回复在法律上并不准确，指出 Flock 作为实际收集并处理身份信息的实体应履行 CCPA 义务，目前正考虑通过法律途径解决。
