  2. TIL: You can make HTTP requests without curl using Bash /dev/TCP
    comments

    在没有 curl 或 wget 的精简容器镜像中，可以利用 Bash 内置的 /dev/tcp 网络重定向功能，通过手动构造 HTTP/1.1 请求来实现基础的连通性检查。该技巧利用 Bash 内部处理的套接字映射而非真实磁盘文件，允许开发者在调试时直接向指定主机和端口读写数据。需要注意该方法仅支持明文传输且不支持 TLS 加密，且依赖于 Bash 编译时的特定网络重定向选项，在 dash 或 zsh 中无法使用。

  3. Apple is about to make Hide My Email useless
    comments

    Apple 宣布将“通过 Apple 登录”和“隐藏邮件地址”的别名域名统一迁移至 @private.icloud.com。这一变化使得第三方服务可以轻易地识别并屏蔽此类匿名别名，而不会误伤正常的 iCloud 邮箱，可能导致这些地址像临时邮箱一样被普遍拒收。此举被认为严重削弱了 iCloud 的隐私保护属性，剥夺了别名地址此前因与主域名混淆而享有的“合理推诿性”。
