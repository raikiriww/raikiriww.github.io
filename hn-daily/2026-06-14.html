Daily Hacker News for 2026-06-14

The 10 highest-rated articles on Hacker News on June 14, 2026 which have not appeared on any previous Hacker News Daily are listed below, with Chinese summaries for quick reading.

  1. Firewood Splitting Simulator
    comments

    这是一款名为 Firewood 的在线劈柴模拟器，属于 screen.toys 系列网页趣味交互工具，由开发者 shapiro500 制作。用户可以通过拖动鼠标调整视角，并点击虚拟斧头将木材劈开，旨在提供一种极简且解压的互动视听体验。

  2. New pancreatic cancer drug might open the door to much longer survival times
    comments

    新药 daraxonrasib 在治疗胰腺癌方面取得重大突破，临床结果显示其将患者的中位生存期从 6.7 个月延长至 13.2 个月，几乎翻了一番。该成果在 5 月底的芝加哥肿瘤学会议上引发数千名科学家起立鼓掌，被视为攻克此类极具挑战性癌症的关键进展。这可能是首款揭示癌症“总开关”并开启全新治疗类别的药物，为显著延长患者生命带来了希望。

  3. GLM 5.2 Is Out
    comments

    智谱 AI 正式发布了其迄今为止最强大的开源模型 GLM-5.2，旨在通过激进的开放立场应对外部技术限制，强调前沿智能应服务于所有开发者。该模型支持真正可用的 100 万（1M）超长上下文窗口，在长程任务处理和复杂智能体（Agent）应用构建方面具有显著优势。目前 GLM-5.2 已面向编程计划用户开放，相关 API 预计将于下周正式上线。

  4. AI coding at home without going broke
    comments

    居家进行 AI 编程主要有自建本地服务器、租用开源模型 API 以及利用顶级模型订阅额度三种方式。文章建议大多数开发者通过 API 租赁以规避硬件过时风险，并提倡采用混合模式：由顶级模型进行高层次思考与规范编写，廉价的开源模型则负责具体机械实现的填充。这种规格驱动的开发策略能让个人以极低成本实现以往庞大工程团队的产出效率。

  5. Police officer investigated for using AI to 'create evidence' in multiple cases
    comments

    英国德比郡一名警官因涉嫌在多起案件中使用人工智能（AI）“伪造证据”而接受调查，目前已被调离一线岗位。皇家检察署（CPS）正配合警方联系受影响的辩护团队和法院，评估虚假证据对司法程序的影响。此前西米德兰兹警方也曾因依赖AI生成的虚构比赛信息而误导决策，这些事件在PoliceAI中心成立之际，引发了对警务系统应用AI真实性与合规性的高度关注。

  6. Honda Civics and the Evil Valet
    comments

    Eric McDonald 对 2021 款本田 Civic 车载中控的逆向工程研究取得突破，发现该系统使用公开的 AOSP 测试密钥验证 USB 更新，存在被称为“EvilValet”的物理访问安全漏洞。攻击者通过车内 USB 接口即可签署并安装自定义更新包，实现任意代码执行甚至获取根权限。作者为此发布了 ota-builder 和 apk-rebuilder 等自动化工具，并呼吁社区协助进行版本适配、工具链优化及 AIDL 接口映射等后续工作。

  7. Free SQL→ER diagram tool, runs in the browser, nothing uploaded
    comments

    SQL to ER Diagram 是一款免费开源的在线工具，支持将 PostgreSQL、MySQL、SQLite 及 SQL Server 的 SQL 模式实时转换为交互式实体关系图（ERD）。该工具完全在浏览器本地运行，无需上传数据或注册账户，有效保障了数据库架构的隐私安全。用户可以自由拖拽布局、添加注释，并将生成的图表导出为 PNG、SVG 格式或通过编码链接进行分享。

  8. How to earn a billion dollars
    comments

    保罗·格雷厄姆通过指数增长的数学模型论证了在不作弊的前提下通过创业赚取10亿美元的可行性，强调财富积累的核心在于增长率和持续时间。他指出创业成功的关键是对用户需求的深刻同理心而非剥削，并建议创始人通过满足自己和朋友的实际需求来预测未来市场。他认为最伟大的创业点子往往源于最初看起来荒谬、旨在解决兴趣的合作项目，而非刻意搜寻的商业计划。

  9. Not everyone is using AI for everything
    comments

    Gabriel Weinberg 通过多项数据指出，尽管媒体宣称 AI 已无处不在，但实际调查显示仅约三分之一的美国成年人积极使用 AI，大部分民众仍持偶尔尝试或完全回避的态度。Gen Z 的采用率已趋于停滞，民众对隐私泄露、失业风险和虚假信息的深度担忧导致负面情绪显著上升。作者将 AI 消费比作肉类摄入，认为用户存在从深度依赖到彻底拒绝的连续光谱，企业应通过提供隐私安全、功能可选的差异化产品来回应这些现实顾虑。

  10. Show HN: Kage – Shadow any website to a single binary for offline viewing
    comments

    Kage（影）是一个开源的网站克隆工具，它利用无头 Chrome 渲染页面并抓取 DOM 快照，随后彻底剥离所有 JavaScript 脚本以实现安全、无追踪的离线浏览。该工具会自动本地化 CSS、图像和字体资源，并支持将镜像站点打包成标准的 ZIM 归档或生成内置查看器的独立二进制文件，方便跨平台分发和永久保存。
