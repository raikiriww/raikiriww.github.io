The 10 highest-rated articles on Hacker News on April 11, 2026 which have not appeared on any previous Hacker News Daily are listed below, with Chinese summaries for quick reading.

  1. Starfling: A one-tap endless orbital slingshot game in a single HTML file
    comments

    Starfling 是一款基于单 HTML 文件开发的极简太空竞技游戏，玩家通过单次点击在星球间进行轨道弹射以挑战飞行极限。游戏玩法高度依赖于对时机的精准掌控，通过在旋转轨道上适时释放来确保飞向下一颗星体并收集星星。除了基础的分数追踪和高分纪录，游戏还集成了广告复活机制与移除广告的内购功能，并计划在 iOS 和 Android 平台上线。

  2. CPU-Z and HWMonitor compromised
    comments

    CPUID 官方网站在 4 月 9 日至 10 日期间因后端 API 遭入侵，导致 CPU-Z 和 HWMonitor 的部分下载链接在约 6 小时内被随机篡改为恶意程序。虽然官方签名的原始软件未受直接篡改，但攻击者通过伪造安装程序分发含有恶意 CRYPTBASE.dll 的窃密木马，利用 PowerShell 在内存中运行以规避检测并尝试获取浏览器凭据。目前该漏洞已修复，此次事件警示攻击者即使不接触源代码，也能通过入侵后端分发环节造成严重的供应链安全威胁。

  3. Industrial design files for Keychron keyboards and mice
    comments

    Keychron 在 GitHub 上公开了其旗下 100 多款键盘和鼠标型号的工业设计文件，涵盖 Q、K、V 及 M 系列等 92 个设备型号，提供 730 余个 STEP 和 DXF 等格式的生产级 CAD 资源。该项目旨在降低硬件开发门槛，支持用户研究生产流程、重构外壳定位板或设计第三方配件。项目采用源码可用许可，允许个人与教育用途，并明确豁免了原创兼容配件的商业化限制，但不允许仿制售卖整机或使用其商标进行品牌宣传。

  4. AI assistance when contributing to the Linux kernel
    comments

    Linux 内核文档明确了 AI 辅助开发的指导规范，要求相关贡献必须符合 GPL-2.0 许可并遵循标准流程。准则强调 AI 代理严禁签署 Signed-off-by 标签，必须由人类提交者审查代码并签署开发者来源证书（DCO）以承担全部责任。此外，AI 参与的贡献需通过 Assisted-by 标签进行详细标注，记录所使用的 AI 名称、模型版本及专业分析工具。

  5. Chimpanzees in Uganda locked in eight-year 'civil war', say researchers
    comments

    乌干达 Ngogo 地区的黑猩猩群体在维持长期团结后，于 2015 年分裂为“西方”和“中央”两个敌对阵营，并陷入了长达八年的暴力“内战”。研究显示，西方群体通过数十次针对性袭击导致中央群体至少 24 只黑猩猩死亡，其中包括多只成年雄性和幼崽。研究人员认为，群体规模过大引发的资源竞争以及雄性间的繁殖竞争是导致这场持续性同类残杀的主要诱因。

  6. Installing every* Firefox extension
    comments

    作者通过调用 Firefox 插件商店 API 抓取并分析了约 8.4 万个扩展，利用日期筛选和类别搜索突破了单次查询 3 万条的分页限制，实现了 99.94% 的覆盖率。分析揭示了插件生态的乱象，包括体积达 196MB 的音频类插件、请求数千项权限的 SEO 垃圾内容，以及通过同形异义词伪装进行虚拟货币窃取的钓鱼程序。研究还识别出大量由 AI 生成的低质量“全能搜索”类潜在有害程序（PUA），其受众规模超 70 万人，相关数据集已托管至 Hugging Face 供社区研究。

  7. Filing the corners off my MacBooks
    comments

    作者针对 MacBook 底部边缘及凹槽处过于锋利导致手腕不适的问题，分享了利用锉刀和砂纸对手体进行圆润化打磨的改装经验。操作中通过胶带封堵键盘与扬声器防止铝屑侵入，并依次使用粗锉及 150/400 目砂纸进行精细加工以确保触感顺滑。作者提倡根据个人需求对生产力工具进行定制化物理修改，并表示将继续在未来的设备上沿用此做法。

  8. Artemis II safely splashes down
    comments

    2026年4月10日，NASA阿耳忒弥斯2号（Artemis II）载人飞船在完成10天绕月任务后，于圣迭戈附近的太平洋海域精准溅落。四名宇航员安全通过大气层再入的高温考验，由美国海军协助顺利出舱并送往回收舰，标志着该历史性探月任务圆满结束。NASA官员称此举开启了深空探索的新纪元，特朗普总统随后也对机组人员表示祝贺并展望了未来的火星计划。

  9. France's government is ditching Windows for Linux, says US tech a strategic risk
    comments

    法国政府宣布将放弃 Windows 转向 Linux 操作系统，旨在减少对非欧盟（尤其是美国）专有技术的依赖并实现数字主权。跨部委数字局（DINUM）已要求各部委在今秋前完成依赖项梳理并制定退出计划，以夺回对数据、基础设施和战略决策的控制权。此项战略转型将优先考虑 openSUSE 和 LibreOffice 等源自欧盟内部的开源解决方案，以应对微软等公司在授权模式及数据安全方面的潜在风险。

  10. Small models also found the vulnerabilities that Mythos found
    comments

    AISLE 的研究显示，参数量仅 3.6B 的开源小模型在检测 FreeBSD 和 OpenBSD 等重大 0-day 漏洞上表现出色，甚至在特定安全推理任务中超越了 Anthropic 的 Mythos 等顶级模型。这证明 AI 网络安全能力具有“锯齿状”特征，并不直接取决于模型规模，其核心护城河在于由安全专家构建的编排系统和验证架构。这种现状改变了防御经济学，使得利用大量低成本模型进行广泛扫描，比依赖昂贵的单一闭源模型更为高效且专业。
