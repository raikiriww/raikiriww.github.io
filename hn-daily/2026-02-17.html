  4. 04
    SkillsBench: Benchmarking how well agent skills work across diverse tasks

    文章提出 SkillsBench，用于衡量 Agent Skills 在 11 个领域共 86 个任务中的实际增益，并为每个任务配套精心编排的 Skills 与确定性验证器。评测覆盖无 Skills、编排 Skills 和自生成 Skills 三种条件、7 种代理模型配置与 7308 条轨迹，结果显示编排 Skills 平均通过率提升 16.2 个百分点，但不同领域差异显著且 84 个任务中有 16 个出现负向变化。自生成 Skills 平均无收益，2-3 模块的聚焦 Skills 优于全面文档型说明，小模型在使用 Skills 后可接近大模型的表现。
