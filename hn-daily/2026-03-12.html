    Making WebAssembly a first-class language on the Web

    文章回顾 WebAssembly 自 2017 年发布以来的能力扩展，包含共享内存、SIMD、异常处理、尾调用、64 位内存与 GC 等特性，但仍存在与 Web 平台集成不足的问题。作者指出 WebAssembly 只能通过 JavaScript 间接访问 Web API，加载与调用流程复杂，需要大量胶水代码，增加构建与运行成本并降低开发体验。这种“二等语言”地位使其更适合大型项目，普通开发者更倾向使用 JavaScript，文章进一步讨论了工具链支持、文档与性能等方面的障碍，并引出组件化方案的潜力。

    BitNet: Inference framework for 1-bit LLMs

    该项目发布了 bitnet.cpp，作为 1-bit LLM 的官方推理框架，提供面向 CPU 与 GPU 的优化内核，强调在多硬件平台上实现快速且无损的推理。文中给出在 ARM 与 x86 上的加速与能耗下降数据，并指出可在单 CPU 上运行 100B 规模的 BitNet b1.58，速度达到每秒约 5-7 tokens。项目还新增并行内核、可配置分块和嵌入量化以继续提速，并提供技术报告与优化指南供进一步参考。

  6. 06
    Show HN: s@: decentralized social networking over static sites

    该页面介绍 s@（sAT Protocol）这种基于静态站点的去中心化社交协议，用户以域名作为身份，在个人静态网站上保存加密 JSON 数据，由浏览器客户端聚合动态并发布内容，不依赖中间服务器或中继。内容采用端到端加密与密钥封装机制：每位用户生成 X25519 密钥对，帖子用对称密钥加密，并按关注者分别封装密钥，实现只对互相关注的朋友可见。协议定义了发现文档、关注列表、帖子索引与存储结构，以及发帖、解密与时间线合并的流程，强调只适用于小范围真实社交。
