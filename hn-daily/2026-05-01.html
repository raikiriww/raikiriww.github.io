The 10 highest-rated articles on Hacker News on May 01, 2026 which have not appeared on any previous Hacker News Daily are listed below, with Chinese summaries for quick reading.

  1. Opus 4.7 knows the real Kelsey
    comments

    作者 Kelsey Piper 发现 Claude Opus 4.7 具备极强的作者去匿名化能力，仅凭 125 字左右的匿名文本、非典型文风草稿或十几年前的旧作，即可精准锁定其身份。这种能力源于 AI 对微观文风特征的敏锐捕捉，即使是在无账号信息关联的情况下，依然能通过对比海量公开语料识别出特定作者。该现象预示着互联网匿名性的实质性终结，尤其是对于有大量公开写作记录的用户，未来通过 AI 追溯职场匿名评价或私人发帖身份将变得异常容易。

  2. How an oil refinery works
    comments

    石油精炼厂通过常压和减压蒸馏利用沸点差异分离原油组分，并运用催化裂化与热裂化技术将重质分子转化为高价值的汽油及化工原料。此外，精炼过程还涉及催化重整、异构化和加氢处理等多种化学工艺，旨在优化分子结构并去除硫等杂质，最后通过大规模罐区设施完成油品存储。

  3. Spain's parliament will act against massive IP blockages by LaLiga
    comments

    西班牙众议院委员会通过了一项由 PSOE 和 ERC 达成的倡议，计划修订《数字服务法》以限制西甲联盟（LaLiga）反盗版诉讼导致的过度 IP 封锁。该提案要求在执行法院判决时遵循技术比例原则，防止公共服务平台等合法第三方网页因共享基础设施而遭到误封。此次投票确立了保护信息权和数字创新的改革框架，旨在防止私人利益干扰互联网的正常运作。

  4. Shai-Hulud Themed Malware Found in the PyTorch Lightning AI Training Library
    comments

    深度学习框架 lightning 的 PyPI 版本 2.6.2 和 2.6.3 遭遇“沙虫”（Shai-Hulud）主题供应链攻击，混淆的恶意代码在模块导入时即自动运行。该程序通过多渠道窃取本地、GitHub 及三大云平台（AWS、Azure、GCP）的敏感凭据，并能利用获取的令牌篡改 npm 包以实现跨生态蔓延。攻击者还针对 Claude Code 和 VS Code 植入持久化钩子，并利用 GitHub 提交搜索作为隐蔽的数据传输机制，严重威胁开发环境及 CI/CD 流水线安全。

  5. How Mark Klein told the EFF about Room 641A [book excerpt]
    comments

    退休AT&T技术人员Mark Klein向电子前哨基金会（EFF）提供关键物证，揭示了NSA在旧金山AT&T设施内设立秘密房间Room 641A，通过分光器大规模复制互联网骨干网流量进行非法监控。该证据确凿地证明了电讯巨头如何协助政府实施无差别监听，被专家形容为针对全美的“国家级窃听”。EFF随后据此发起法律诉讼，尽管面临持有机密信息的潜在法律风险，仍致力于揭露并终止这一违宪的监控项目。

  6. For Linux kernel vulnerabilities, there is no heads-up to distributions
    comments

    CVE-2026-31431（CopyFail）是一个自2017年内核4.14版本引入的严重本地提权漏洞，由于API变更导致回溯移植困难，目前6.12等多个LTS版本仍未修复。Gentoo开发者透露，除非报告者主动向发行版邮件列表报送，否则内核安全漏洞通常不会向各分发版提供预警，导致其在漏洞公开时无法提前防御。目前针对旧版本内核的临时应对措施是通过补丁禁用authencesn加密模块以阻断攻击路径。

  7. LinkedIn scans for 6,278 extensions and encrypts the results into every request
    comments

    LinkedIn 被曝在用户不知情的情况下秘密扫描浏览器插件，扫描列表已从 2017 年的 38 个激增至 2026 年的 6278 个，且未在隐私政策中披露。该技术通过探测插件的特定资源文件，将软件清单与用户的真实职业身份直接绑定，从而推断其求职意向、政治立场及所在企业的内部工具。相关数据经加密后嵌入每个 API 请求发送，并与第三方脚本共享以实现跨站追踪，目前德国巴伐利亚检察机关已针对其违反 DMA 规定及非法采集指纹的行为启动刑事调查。

  8. Can I disable all data collection from my vehicle?
    comments

    Rivian车辆联网是核心功能，用户虽可禁用数据收集，但这将导致导航、主动车道居中及OTA远程更新等关键功能受限。加拿大版车辆可通过“数据与隐私”设置自行关闭，而其他地区车辆则需联系官方服务中心通过预约禁用车载eSIM。此外，禁用连接不会自动终止Connect+等订阅服务，用户需另行手动取消。

  9. Grok 4.3
    comments

    xAI 推出的 Grok 4.3 模型支持 100 万 token 超长上下文，具备推理、函数调用及结构化输出能力。API 基础定价为每百万输入 token 1.25 美元、输出 2.50 美元，利用缓存技术可将输入成本降至 0.20 美元。该模型提供每分钟 1800 次请求的高速率限制，并针对超过 200K 窗口的超长上下文请求执行差异化计费。

  10. Show HN: WhatCable, a tiny menu bar app for inspecting USB-C cables
    comments

    WhatCable 是一款专为 Apple Silicon Mac 开发的开源工具，能以通俗易懂的语言实时显示 USB-C 线缆的传输速度（最高 80Gbps）、充电功率（最高 240W）及 e-marker 芯片等详细规格。它利用 IOKit 服务识别充电瓶颈、电源 PDO 列表及连接设备身份，解决了 USB-C 线缆外观相似但内部标准混乱的痛点。该工具提供菜单栏应用及命令行界面，支持通过 Homebrew 安装，但由于底层驱动限制暂不支持 Intel 机型。
