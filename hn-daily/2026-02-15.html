    News publishers limit Internet Archive access due to AI scraping concerns

    多家新闻机构担心 AI 爬虫通过 Internet Archive 及其 Wayback Machine、API 取得结构化内容，开始限制相关访问。The Guardian 采取排除 API 与过滤文章页等措施，纽约时报与多家媒体在 robots.txt 中对 archive.org 的机器人进行硬阻断；机构强调是防止未经授权的抓取与知识产权外泄。报道还提到部分出版集团在样本中大规模屏蔽 Internet Archive 及 Common Crawl 等爬虫，并引用互联网档案馆的回应，称限制图书馆会削弱公众获取历史记录的能力。

    Oat – Ultra-lightweight, zero dependency, semantic HTML, CSS, JS UI library

    该库主打超轻量与零依赖，仅需引入约 8KB 的 CSS/JS 就能获得常用 UI 组件与元素，强调语义化 HTML，尽量不使用类名并用原生标签与 ARIA 角色直接样式化。部分动态组件采用 WebComponents，提供键盘导航与无障碍支持，并可通过少量 CSS 变量快速定制主题，data-theme="dark" 可启用内置暗色方案。作者强调反对繁重框架与生态锁定，希望用纯 CSS/JS 的长期可维护、标准化方案实现类似 shadcn 风格的外观与体验。

  7. 07
    I fixed Windows native development

    作者指出在 Windows 上要求安装 Visual Studio 会把维护者变成安装器的技术支持，组件繁杂、安装缓慢且版本不可控，导致构建环境难以复现、迁移成本高。为此他开发了开源 CLI 工具 msvcup，直接解析微软公开的组件清单并从 CDN 下载最小化的 MSVC 工具链与 SDK，以版本化目录隔离安装。通过脚本即可在几分钟内完成安装并生成自动环境，支持交叉编译、锁文件与可重复构建，且无需 GUI 安装器和全局污染。该方案已在真实项目和 CI 中应用，显著降低 Windows 原生开发的门槛，但若需要完整 IDE 仍需官方安装器。

    EU bans the destruction of unsold apparel, clothing, accessories and footwear

    欧盟委员会于2026年2月9日在《可持续产品生态设计法规》(ESPR)框架下通过新措施，禁止销毁未售出的服装、配饰和鞋类，并要求企业披露被作为废弃物处理的未售消费品信息。该举措旨在减少浪费和环境损害，文中估算欧洲每年有4%-9%的未售纺织品被销毁，产生约560万吨二氧化碳排放。禁令及豁免条款将自2026年7月19日对大型企业生效，中型企业预计2030年跟进，信息披露的统一格式自2027年2月起适用。
