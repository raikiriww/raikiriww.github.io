    Yann LeCun raises $1B to build AI that understands the physical world

    前 Meta 首席 AI 科学家 Yann LeCun 共同创办的巴黎初创公司 AMI 宣布融资超过 10 亿美元，估值约 35 亿美元，目标是构建基于“世界模型”的新一代 AI 系统。LeCun 认为人类推理主要扎根于物理世界，单靠扩展大语言模型无法达到人类级智能，AMI 将面向制造、生物医药与机器人等拥有大量数据的行业提供可记忆、可推理、可规划且可控安全的系统。公司计划从一开始就全球化布局，设有巴黎、蒙特利尔、新加坡和纽约办公室，并由多位前 Meta 研究负责人及前 Nabla CEO 等联合创始人领导。

    Agents that run while I sleep

    作者描述让代码代理在自己睡觉时长时间运行，变更落到分支却难以确认是否正确，团队在大量合并与审查负担下信任危机加剧。他认为同一模型写代码又写测试是自我验证，主张回到类似 TDD 的思路：先用自然语言写清可验证的验收标准，再让代理实现并用自动化验证。文中给出前后端验收标准与 Playwright 验证的流程示例，并提出由脚本预检、规划、并行浏览器执行、最终判定组成的工具链，强调先定义“完成”的可观察结果，才能提高代理产出的可信度。

  3. 03
    Cloudflare crawl endpoint

    Cloudflare 在 Browser Rendering 中推出新的 /crawl 端点（开放测试），允许用一次 API 调用从起始 URL 自动发现并抓取整站内容，结果可返回 HTML、Markdown 与结构化 JSON。该端点默认遵循 robots.txt 与 AI Crawl Control，并以异步任务形式运行：提交后拿到任务 ID，再轮询获取处理进度与结果。功能包含抓取范围控制、站点地图与链接双源发现、增量抓取与静态模式等，并适用于免费与付费计划，但不会绕过机器人检测或验证码。

  4. 04
    U+237C ⍼ Is Azimuth

    作者追溯符号 ⍼ 的来源，指出 1950 年 H. Berthold AG 的符号目录将其标为 Azimut/方向角，从而解开长期谜团。对比 1949–1952 年的字样目录与更早的 1946、1909、1900 年目录，发现该符号在 1950 前并未出现，并给出相应页面扫描位置。作者还引用 Mastodon 的观察，说明 ⍼ 形状像六分仪中光线路径与直角符号，并联系 Wikipedia 的六分仪示意图解释其测量方位角的含义。

  5. 05
    Zig – Type Resolution Redesign and Language Changes

    该更新合并了一个约3万行的重构分支，目标是把 Zig 编译器内部的类型解析逻辑改成更直观的设计，并带来多项用户可见变化。编译器现在会更惰性地分析类型字段，未被初始化的类型不会被展开分析，从而避免命名空间用法触发不必要的编译错误。依赖环路的报错信息变得可读，能够指出环路来源与关键位置，便于定位问题。此外，增量编译修复了大量已知问题，显著减少过度分析并提升更新速度，同时还包含若干性能改进与小幅语言调整。

    Temporal: A nine-year journey to fix time in JavaScript

    文章回顾了 Temporal 在 TC39 标准化过程中长达九年的推进历程，解释了 JavaScript 早期沿用 Java Date 设计在可变性、月份计算和字符串解析等方面留下的长期痛点。作者结合 Bloomberg 在跨时区金融系统中的实践，说明生态被迫依赖 Moment.js 等库却带来体积和维护负担，从而推动以不可变类型、时区与历法一等公民、纳秒精度为目标的新方案。文中还梳理了 Bloomberg 与 Igalia 等多方协作、冠军团队扩充以及 Temporal 作为顶层命名空间提供多种类型（如 Temporal.ZonedDateTime）的现状与定位。
