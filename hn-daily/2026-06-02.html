  1. Nvidia RTX Spark
    comments

  2. AI Agent Guidelines for CS336 at Stanford
    comments

    斯坦福大学CS336课程发布了AI助手使用指南，明确AI应扮演“助教”而非“代码生成器”的角色，通过引导性提问、概念解释和建议调试步骤来协助学生。准则严禁AI直接编写Python代码、修改仓库或提供完整方案，旨在确保学生通过自主实现Transformer架构、分词器及分布式训练等核心组件，在“做中学”里真正掌握大模型底层技术。

  3. Age verification for social media, the beginning of the end for a free internet?
    comments

    全球多国正以保护儿童为名推行社交媒体年龄验证，实质上是通过强制身份认证建立政府管控的互联网体系，严重威胁匿名权与数字隐私。这种趋势正从社交平台蔓延至VPN服务及操作系统层面，通过应用商店和OS层面的身份绑定实施内容过滤，引发了严重的言论自律和审查风险。批评者警告，此类政策最终可能导致对用户自主控制设备及开源系统的封禁，标志着自由互联网向全面监控时代的转变。

  5. macOS needs its grid back
    comments

    作者怀念 macOS Leopard 时代支持空间与肌肉记忆的 3x3 虚拟桌面网格，认为 Lion 之后改为水平线性排列是效率的退步。为此他开发了名为 GridLion 的应用，通过轻量级包装原生空间 API 重新实现网格化导航与预览功能，并克服了权限限制与私有 API 调用等障碍。在开发过程中作者虽利用 LLM 快速构建原型，但强调开发者对产品细节的深度关切和打磨才是确保软件价值与良好体验的关键。

  6. Why Janet? (2023)
    comments

    Janet 是一种极简且现代化的 Lisp 方言，其核心指令集仅有 8 条，支持通过 C 编译器生成包含运行时环境且体积小于 1MB 的静态原生可执行文件。它以解析表达式语法（PEG）取代正则表达式，提供了更强大的文本与二进制解析能力，并拥有出色的子进程 DSL 和易于嵌入 C 程序的特性。此外，Janet 具备完善的宏系统与编译期代码执行能力，允许通过程序状态快照实现高效的资产嵌入和数据预处理，是 Lua 或 Bash 的理想替代方案。
