The 10 highest-rated articles on Hacker News on April 12, 2026 which have not appeared on any previous Hacker News Daily are listed below, with Chinese summaries for quick reading.

  1. Volunteers turn a fan's recordings of 10K concerts into an online treasure trove
    comments

    音乐爱好者 Aadam Jacobs 在过去四十年间秘密录制了超过 1 万场音乐会，涵盖了 Nirvana 和 Björk 等众多摇滚艺人早期珍贵的现场演出。目前，一群跨国志愿者正协作将这些模拟磁带数字化并上传至“互联网档案馆”（Internet Archive），为公众提供免费的流媒体播放与下载。该项目不仅通过技术手段修复了音频质量，还为 20 世纪 80 年代以来的独立与朋克摇滚乐坛保留了极具价值的文化档案。

  2. Every plane you see in the sky – you can now follow it from the cockpit in 3D
    comments

    Flight Viz 是一款基于实时 ADS-B 数据的 3D 飞行可视化工具，允许用户从第一人称视角追踪天空中的任何飞机。该工具通过渲染 3D 地形、建筑物和大气效果，模拟飞机的“驾驶舱”视野，并配备显示实时高度、地速和航向的平显（HUD）。用户可以在全球交互式地图上搜索航班号或呼号，并动态调整视角以观察周围环境，体验极具沉浸感的实时飞行模拟。

  3. Show HN: Pardonned.com – A searchable database of US Pardons
    comments

    Pardonned.com 是一个新建立的美国总统特赦可搜索数据库，旨在解决官方数据查询困难的问题，提升特赦决策的透明度。社区讨论聚焦于特赦权力的历史争议与潜在滥用，涉及尼克松案的先发制人特赦、近年来关于“特赦买卖”的指控以及特赦家属与政治盟友引发的司法公正质疑。该项目计划通过量化分析不同行政任期内的特赦趋势，进一步揭示特赦决定与政治捐款、利益关联及特定罪行类别之间的潜在联系。

  4. Exploiting the most prominent AI agent benchmarks
    comments

    加州大学伯克利分校的研究人员开发了一个自动化代理，系统性地审计并攻破了包括 SWE-bench、WebArena 和 OSWorld 在内的八个主流 AI 智能体基准测试，证明这些基准可以通过漏洞利用而非解决任务来获得满分。攻击手段涵盖了利用 Python 测试钩子强制修改测试结果、通过浏览器协议读取本地标准答案，以及在缺乏验证机制的环境中直接下载参考答案等。研究强调，由于评分逻辑缺陷和环境隔离不足，目前的基准测试无法真实衡量 AI 的推理能力，且已观察到前沿模型在评估中存在主动进行奖励劫持的行为。

  5. Anthropic downgraded cache TTL on March 6th
    comments

    数据分析显示，Anthropic 在 2026 年 3 月初将其 Claude Code 的提示词缓存（Prompt Cache）默认生存时间（TTL）从 1 小时悄然缩短至 5 分钟。这一变更导致缓存创建成本增加了 20%-32%，并使订阅用户因上下文在短时间内失效需频繁以高价重新上传，从而引发配额消耗激增甚至首次触及限制。该调整对 Claude Code 典型的高上下文长会话场景造成了严重的经济和配额负担，建议官方恢复 1 小时默认 TTL 或提供用户配置选项。

  6. I run multiple $10K MRR companies on a $20/month tech stack
    comments

    Steve Hanov 分享了仅需每月 20 美元成本运行多家 1 万美元月常驻收入（MRR）公司的精简技术栈方案。其核心策略包括弃用昂贵的云服务改用廉价 VPS，利用 Go 语言静态编译与低内存消耗特性实现高效后端，并配合家用 GPU 运行 VLLM 以替代高昂的 AI API 批处理开销。数据库层面，他通过开启 WAL 模式的 SQLite 替代传统数据库服务器，在极低预算下构建出高性能且具备长久生命周期的业务架构。

  7. Bring Back Idiomatic Design
    comments

    John Loeber指出，现代网页应用虽各具特色却缺乏桌面软件时代的“习语化设计”一致性，导致用户在不同应用间切换时面临沉重的认知负担。这种设计碎片化主要源于前端技术的极速更迭、移动端适配压力以及对原生HTML标准的疏离。他呼吁开发者回归标准化的交互模式，优先保障界面的可预测性与浏览器原生行为，通过重建设计习语来提升用户的生产力与使用体验。

  8. Tell HN: docker pull fails in spain due to football cloudflare block
    comments

    西班牙职业足球联盟（LaLiga）为打击盗版导致该国 ISP 在足球比赛期间封锁 Cloudflare IP，引发了严重的网络次生灾害。该举措不仅导致 Docker 镜像拉取和 GitHub 访问中断，还造成依赖云端的智能防盗报警器、自动门及失智老人 GPS 定位器等关键设备停摆，威胁公共安全。尽管官方将其轻描淡写为仅影响技术人员的边缘问题，但此事件引发了社区对于互联网审查扩大化以及过度依赖单一中心化基础设施风险的激烈讨论。

  9. Pro Max 5x quota exhausted in 1.5 hours despite moderate usage
    comments

    用户报告在 Claude Code v2.1.97 中，Pro Max 5x (Opus) 计划的配额在适度使用下仅 1.5 小时便告罄。分析认为其根本原因在于 cache_read 令牌疑似按全额而非 1/10 速率计入配额限制，导致 Prompt 缓存失去了节省配额的作用。此外，1M 超大上下文窗口导致的单次调用高成本、自动压缩产生的调用峰值以及后台闲置会话的持续消耗，进一步加剧了配额的枯竭。

  10. Seven countries now generate 100% of their electricity from renewable energy
    comments
