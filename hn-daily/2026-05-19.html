Daily Hacker News for 2026-05-19

The 10 highest-rated articles on Hacker News on May 19, 2026 which have not appeared on any previous Hacker News Daily are listed below, with Chinese summaries for quick reading.

  1. Anthropic acquires Stainless
    comments

    Anthropic 宣布收购 SDK 和 MCP（模型上下文协议）工具领域的领先者 Stainless，旨在提升 AI 智能体（Agents）的系统接入能力。Stainless 自 Anthropic API 早期便为其提供多语言官方 SDK 支持，其技术能将 API 规范转化为高性能的原生开发工具。此次收购将深化 Claude 平台在开发者体验和工具连接性方面的布局，进一步拓展智能体与外部数据及系统的交互边界。

  2. Click (2016)
    comments

    这是一款名为“Click click click”的网页游戏，旨在探讨在线个人画像追踪。玩家在游戏中可以尝试解锁 128 项成就，并支持通过特定 URL 保存游戏进度，通过互动形式展示用户的浏览器行为是如何被监控和分析的。

  3. The last six months in LLMs in five minutes
    comments

    Simon Willison 在 PyCon US 2026 上总结了 LLM 过去半年的进展，指出 2025 年 11 月是编码智能体（Coding Agents）因强化学习技术突破而跨越质量门槛、成为日常开发工具的关键拐点。期间以 OpenClaw 为代表的个人 AI 助手（Claws）社区迅速兴起，推动了本地硬件运行 AI 的热潮。近期发布的 Gemma 4 系列及中国 1.5TB 开源大模型 GLM-5.1 进一步证明了开源生态的强劲，在多模态理解与生成性能上均表现出超预期的提升。

  4. CISA Admin Leaked AWS GovCloud Keys on GitHub
    comments

    CISA 一名外包员工在名为“Private-CISA”的公开 GitHub 仓库中泄露了多个高权限 AWS GovCloud 账户凭据、内部 DevSecOps 环境访问权限及大量系统明文密码。调查显示该员工为方便同步文件甚至手动禁用了 GitHub 的机密检测功能，暴露了严重的个人安全意识缺失及机构监管漏洞。安全专家认为此次泄露涉及内部代码仓库等核心资产，是近年来政府部门发生的最严重数据泄露事件之一。

  5. Show HN: Gaussian Splat of a Strawberry
    comments

    Dany Bittel 展示了一个草莓的高质量高斯泼溅（Gaussian Splat）三维模型，该场景通过 90 个视角、每个视角 88 张景深合成照片精细构建。拍摄过程采用了尼康 Z8 全画幅相机与老蛙 180mm 微距镜头，并在 slang-splat 工具中完成模型训练。该作品遵循 CC BY 协议开放下载，作者还免费提供了相关的 COLMAP 数据集供研究使用。

  6. Apple unveils new accessibility features
    comments

    苹果预览了由 Apple Intelligence 驱动的一系列无障碍更新，显著增强了 VoiceOver、放大器和语音控制等核心功能。新推出的功能包括 Apple Vision Pro 用户可通过眼神控制兼容的轮椅，以及利用端侧语音识别技术为个人视频及流媒体内容自动生成实时字幕。此外，visionOS 引入了减少晕车感的车辆运动提示，tvOS 增加了大字体支持，且 iOS 等系统现已兼容索尼 Access 游戏控制器。

  7. I’ve joined Anthropic
    comments

    Andrej Karpathy 宣布正式加入人工智能公司 Anthropic，重返大语言模型（LLM）前沿的研发一线。他认为未来几年是 LLM 发展的关键成型期，并对重回 R&D 领域感到非常兴奋。同时，他明确表示依然保持对教育事业的热情，并计划在未来适当时候恢复相关工作。

  8. I’ve built a virtual museum with nearly every operating system you can think of
    comments

    Virtual OS Museum 是一个集成了 QEMU、VirtualBox 和 UTM 等模拟器的 Linux 虚拟机项目，旨在通过预设的一键启动器，为用户提供即开即用的历史操作系统体验。该项目涵盖了从 1948 年至今的 1700 多个预装系统，包括大型机、工作站、早期家用电脑及移动设备等 570 多种不同型号。开发者 Andrew Warkentin 历时 20 年进行收集与调试，解决了模拟器配置复杂和安装媒介稀缺的难题，让用户能够直接探索从早期 Unix、NeXTSTEP 到各版本 Windows 的完整计算史。

  9. Gemini 3.5 Flash
    comments

    Google 发布 Gemini 3.5 系列模型并首发 3.5 Flash，该模型专为复杂智能体工作流和编程任务设计，在速度上较其他前沿模型提升 4 倍且成本更低。3.5 Flash 具备卓越的多模态理解与交互式 UI 生成能力，已集成至 Gemini 应用、Google Search 及 Antigravity 开发平台。此外，全新的个人 AI 智能体 Gemini Spark 已基于此模型运行，而高性能的 3.5 Pro 预计将于下月推出。

  10. Tesla's lithium refinery discharges 231,000 gallons of polluted wastewater a day
    comments

    特斯拉位于德州罗布斯敦的锂精炼厂被指每日向公共排水沟排放23.1万加仑黑色废水，当地排水管理部门在例行巡检中发现该未标记管道。虽然特斯拉坚称排放符合现有许可且工艺“无酸清洁”，但第三方检测在废水中发现了六价铬（强致癌物）、砷及异常高浓度的锂和锰，显示出与电池加工高度吻合的工业特征。在当地因严重干旱面临用水限制的背景下，此事件暴露了监管部门对锂精炼污染物监测的缺失，引发了对电动汽车供应链所谓“清洁”标准的质疑。
