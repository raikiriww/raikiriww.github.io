  1. My minute-by-minute response to the LiteLLM malware attack
    comments

  2. $500 GPU outperforms Claude Sonnet on coding benchmarks
    comments

    ATLAS 是一个开源的本地代码助手架构，通过约束驱动生成、能量验证及自验证迭代修复等推理侧技术，显著提升了小型本地模型的逻辑能力。在 LiveCodeBench 评测中，该系统配合 Qwen3-14B 冻结模型在单块 500 美元级别的消费级 GPU 上实现了 74.6% 的通过率，性能超越了 Claude 4.5 Sonnet。项目支持完全私有化部署，通过以本地计算时间换取准确率的策略，证明了在无需微调和云端 API 的情况下，普通硬件也能运行高性能的专家级 AI 工具。

  5. A Faster Alternative to Jq
    comments

    jsongrep 是一款受 ripgrep 启发并由 Rust 编写的高性能 JSON 检索工具，专门用于快速搜索而非数据转换。其核心优势在于将查询语句编译为确定有限状态自动机（DFA），通过对 JSON 文档进行单次遍历实现 O(1) 的状态转移，有效解决了 jq 等工具在处理递归下降查询时的性能瓶颈。在针对大规模数据集的基准测试中，其单次遍历且无回溯的搜索特性使其展现出远超传统解释型工具的处理速度。

  9. Anatomy of the .claude/ folder
    comments

    comments
