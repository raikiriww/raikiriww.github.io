Daily Hacker News for 2026-05-13

The 10 highest-rated articles on Hacker News on May 13, 2026 which have not appeared on any previous Hacker News Daily are listed below, with Chinese summaries for quick reading.

  1. Linux gaming is faster because Windows APIs are becoming Linux kernel features
    comments

    Linux 在 Steam 用户中的份额已突破 5%，其游戏性能的提升正从 Wine 和 Proton 等翻译层深入到内核级别。新引入的 NTSYNC 内核驱动程序直接实现了 Windows 特有的同步机制，使 Linux 能够原生响应游戏协调所需的 API 调用。相比此前的 fsync 等过渡方案，NTSYNC 通过精确匹配 Windows 行为，有效解决了特定游戏中的卡顿和死锁等底层兼容性问题，目前已在 Valve 的 SteamOS 中正式上线。

  2. Why senior developers fail to communicate their expertise
    comments

    资深开发人员往往专注于复杂性管理以保障系统稳定性，而业务部门则更看重通过快速迭代来降低市场不确定性，这种核心关注点的错位是导致沟通失败的根源。资深开发者应学会从降低不确定性的角度描述方案，通过“尝试更快路径”的策略在满足业务速度需求的同时，发挥其精简和复用代码的专业价值。在 AI 时代，建议将系统解耦为追求实验速度的“速度版”和由资深开发者担任“编辑”进行稳固的“规模版”，从而在利用 AI 提速的同时确保系统的长期可维护性。

  3. The Future of Obsidian Plugins
    comments

    Obsidian 推出全新的社区（Obsidian Community）站点与开发者仪表盘，为超过 4000 个插件和主题提供更高效的搜索、发现及管理平台。系统引入了全版本自动化审查机制以加强安全漏洞与代码质量监控，并配套推出安全评分卡、权限披露及作者验证标识。开发者现可通过新工具实现快速提交与状态追踪，官方同时明确了插件付费模式的分类标签，并计划未来支持团队私有插件的分发管理。

  4. Show HN: Needle: We Distilled Gemini Tool Calling into a 26M Model
    comments

    Needle 是一个从 Gemini 3.1 蒸馏而来的 2600 万参数超小型函数调用模型，采用“简单注意力网络”架构，专为手机、手表和眼镜等嵌入式设备设计。该模型在单次工具调用任务上的表现优于规模更大的 FunctionGemma-270m 等模型，在特定硬件上预填充速度可达 6000 tokens/sec。目前该项目已完全开源权重与数据集生成方法，并提供 Web UI 供用户在本地进行工具测试及一键式微调。

  5. CERT is releasing six CVEs for serious security vulnerabilities in dnsmasq
    comments

    CERT 发布了 6 个针对 dnsmasq 长期存在的严重漏洞的 CVE 编号，补丁已在 2.92rel2 稳定版中发布并已预先披露给相关厂商。开发者 Simon Kelley 表示，这些漏洞的发现源于 AI 驱动的安全研究带来的大量报告，给漏洞分拣和修复工作带来了极大挑战。目前 2.93rc1 版本已推出以进一步整合修复，官方建议用户尽快测试并升级以应对 AI 自动化漏洞挖掘带来的持续压力。

  6. How to make your text look futuristic (2016)
    comments

    Dave Addey 在《Typeset In The Future》中总结了营造字体“未来感”的六大核心规则：应用斜体、结合曲线与棱角、添加 V 型切口、合并字符、移除冗余笔画，以及叠加金属拉丝质感、蓝光和星空背景。文章通过分析《银翼杀手》、《星球大战》、《星际迷航》等经典科幻影视作品的 Logo，揭示了这些视觉修饰手段及 Eurostile 等字体如何共同构建出跨越时代的未来主义审美。

    comments

  8. I moved my digital stack to Europe
    comments

    为了追求数字主权和数据隐私，作者将其核心技术栈从美国服务迁移至欧洲，以确保数据留在法律可预测的辖区内。主要变动包括以自建 Matomo 取代 Google Analytics，将计算和存储转至 Scaleway 与 OVH，并采用 Mistral 替代 OpenAI，从而在实现 GDPR 合规的同时降低环境足迹。虽然在 CDN、支付和 GitHub 等具有强网络效应或高迁移门槛的领域暂留了部分美国工具，但整体迁移显著提升了基础设施的自主性。

  9. Leaving GitHub for Forgejo
    comments

    作者 Jorijn 将代码从 GitHub 迁移至自托管的 Forgejo 实例，主因是追求数字主权而非单纯回避其频繁的宕机故障。他强调 GitHub 在并入微软 CoreAI 部门后已失去独立性，且 Copilot 默认将用户数据用于 AI 训练并缺乏仓库级退出机制，加之美国 FISA 702 等法律带来的隐私风险，使其不再适合托管核心代码。这一决策与荷兰政府近期的选择一致，后者同样因 Forgejo 的完全开源属性和非营利社区治理模式，将其选为官方代码托管平台以确立数字自主权。

  10. Setting up a free *.city.state.us locality domain (2025)
    comments

    美国公民或组织可以免费申请形如 *.city.state.us 的地域域名（Locality Domains），该体系自1992年起建立并由政府合同维护。申请过程包括：先通过授权列表找到所在地区的注册商联系方式，利用 Amazon Lightsail 获取支持非顶级域名的免费名称服务器，随后填写并向注册商发送《临时 .US 域名模板》邮件。一旦注册成功，用户即可在 Lightsail 中配置 DNS 记录，将域名指向 GitHub Pages 等 Web 托管服务。
