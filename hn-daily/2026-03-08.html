  3. CasNum
    comments

    CasNum 是一个用圆规直尺作图来实现任意精度运算的库，把数字表示为平面点，并用作图构造来完成加减乘除以及部分逻辑运算。项目还把这些几何构造嵌入到 Game Boy 模拟器的 ALU 中，提供示例与运行方式，并带有作图可视化查看器。作者强调这是一种偏“几何计算”的实验性实现，性能依赖缓存且资源开销大，适合用于教学、演示或概念验证。

  4. Put the zip code first
    comments

  7. Apple's 512GB Mac Studio vanishes, a quiet acknowledgment of the RAM shortage
    comments

    文章指出，尽管苹果近期产品发布看似常规，但其悄然下架了顶配 M3 Ultra Mac Studio 的 512GB 内存选项，反映出 AI 驱动的内存与存储供应紧张已开始影响配置供给。与此同时，256GB 配置价格从 1,600 美元涨到 2,000 美元，而苹果支持站点的规格页仍保留 512GB 选项，商店页面却已删除。作者强调，苹果很少直接移除在售配置，通常只会拉长交期，这次变化因此格外显眼。

  8. FrameBook
