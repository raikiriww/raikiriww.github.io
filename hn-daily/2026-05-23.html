Daily Hacker News for 2026-05-23

The 10 highest-rated articles on Hacker News on May 23, 2026 which have not appeared on any previous Hacker News Daily are listed below, with Chinese summaries for quick reading.

  1. Antigravity 2.0 Tops the OpenSCAD Architectural 3D LLM Benchmark
    comments

    ModelRift 发布了针对 OpenSCAD 建筑 3D 建模的大模型基准测试，对比了 Google Antigravity 2.0、Codex 5.5 及 Claude 等工具在构建罗马万神殿任务中的表现。Google Antigravity 2.0 搭载 Gemini 3.5 Flash High 夺得自主建模冠军，其不仅搜索了真实参数，还唯一实现了复杂的穹顶藻井细节。测试表明 OpenSCAD 的纯文本参数化特性非常适合 LLM 空间推理，其中 Cursor 虽然运行最快但质量垫底，而 ModelRift 通过视觉标注的人机协作模式有效提升了模型的修改精度。

  2. Deno 2.8
    comments

    Deno 2.8 发布了迄今最大的次要版本更新，引入了 import defer 模块加载方式，并新增了包括 deno transpile、deno pack、deno ci 和 deno why 在内的六个实用子命令。该版本显著增强了 Node.js 兼容性，测试通过率从 42% 提升至 76.4%，同时通过并行解析等优化使 npm 冷安装速度加快了 3.66 倍。此外，新版本还支持 Chrome DevTools 网络调试，并在 node:http 吞吐量和 base64 处理性能上实现了成倍增长。

  3. DeepSeek makes the V4 Pro price discount permanent
    comments

    DeepSeek 官方宣布 deepseek-v4-pro 模型的 API 永久定价将调整为原价的 1/4（即 75% 优惠后的价格），其中输出价格仅为 $0.87/1M tokens。同时，所有模型的输入缓存命中价格也已降至首发价的 1/10，显著降低了大规模推理成本。此外，旧有的 deepseek-chat 和 deepseek-reasoner 名称将被弃用，由支持 1M 上下文的 V4 系列模型及其特定模式取代。

  4. U.S. researchers face new restrictions on publishing with foreign collaborators
    comments

    美国NIH和NASA近期对受资助科学家与外国合作者共同发表论文实施了前所未有的限制，要求即便研究完全在美国境内完成，也需就外国机构关联的合著者身份提前申请许可。NIH被指私下要求研究人员从进展报告中删除含外国合著者的文章，而NASA则加强了对《沃尔夫修正案》的执行力度，甚至将不涉及政府资金流出的对华学术合著视为潜在违规。由于相关机构缺乏正式公开指导且采取个别通知的“碎片化”沟通方式，引发了学术界对评估不公及排外倾向的严重担忧，专家警告此举可能导致国际科研协作中断并损害整体科学产出。

  5. Bun support is now limited and deprecated
    comments

    音视频下载工具 yt-dlp 宣布限制并弃用对 Bun 运行时的支持，将支持范围缩小至 1.2.11 到 1.3.14 版本。此举旨在修复旧版本中存在的安全隐患，并表达了对 Bun 切换到 Rust 库后转向“氛围驱动”开发模式所带来的维护和兼容性风险的担忧。开发团队保留未来因负担过重而完全停止支持 Bun 的权利。

  6. Microsoft starts canceling Claude Code licenses
    comments

    微软计划在 6 月 30 日前终止内部数千名开发者对 Anthropic AI 编程工具 Claude Code 的使用授权，并强制其转向 GitHub Copilot CLI。尽管该工具在 Windows 和 Microsoft 365 等部门备受欢迎，但微软出于整合内部命令行代理工具及在财年结束前降低运营成本的考虑，决定停止该项试点并统一使用自家产品。

  7. Project Glasswing: An Initial Update
    comments

    Anthropic发布Glasswing项目首月进展，利用Claude Mythos Preview模型协助合作伙伴发现逾万个高危漏洞，将漏洞发现效率提升了十倍以上。该项目同时识别出开源软件中数千个严重漏洞，揭示了当前安全瓶颈已由寻找漏洞转向人工验证与修复的速度。由于Mythos级别模型显著缩短了漏洞发现与利用的周期，报告呼吁开发者和防御者必须加快补丁发布及部署，并强化网络基础防御措施。

  8. Green card seekers must leave U.S. to apply, Trump administration says
    comments

    特朗普政府宣布重大政策变更，要求大多数绿卡申请人必须回到母国通过领事程序办理，仅在“非凡情况”下允许在美境内申请身份调整。此举将直接影响包括留学生、美国公民配偶及外籍劳工在内的数十万合法入境者，可能导致严重的家庭分离及海外领事馆的审理压力。法律专家认为这是收紧合法移民政策的重大升级，预计将面临广泛的法律挑战。

  9. Shipping a laptop to a refugee camp in Uganda
    comments

    作者为帮助在乌干达难民营远程学习的刚果学生 Django，尝试从澳大利亚寄送旧 MacBook，却遭遇了锂电池航空禁运、高昂的商业货运费及极其复杂的跨境官僚审查。Django 为获取清关所需的税号（TIN），在资源匮乏的营地与政府机构间艰难周旋，并克服了暗示索贿、网络故障以及海关因缺少原始收据而扣押设备等重重阻碍。最终，尽管物流跟踪失效且总花费高达 426 澳元（接近电脑价值），这台设备在历经数周的波折后终于接近交付。

  10. Texas woman arrested for Facebook post about town water quality
    comments

    德克萨斯州特立尼达镇居民 Jennifer Combs 因在 Facebook 发帖称当地水质导致居民住院而遭警方逮捕，被控以通常针对虚假炸弹威胁的“虚假警报”重罪。尽管该市承认供水管道老化严重且随后发布了煮沸用水通知，但警方坚称其传播的住院信息纯属虚假。Combs 目前已提起联邦诉讼，指控当地政府通过报复性执法来压制公众对水安全问题的合法言论。
