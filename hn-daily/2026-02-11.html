  4. 04
    The Day the Telnet Died

    文章记录 GreyNoise 观测到 2026-01-14 21:00 UTC 全球 Telnet 流量出现断崖式下跌，小时会话量从约 7.4 万降到 2.2 万并在数小时内稳定在更低水平，日均量较基线长期下降约 59%。多个大型 AS 和国家的端口 23 流量归零，而部分云厂商影响较小，作者据此推测可能是北美骨干或一级运营商实施了端口 23 过滤，并指出与 2026-01-20 公布的 GNU Inetutils telnetd 漏洞 CVE-2026-24061 时间接近但因果尚无法证明。文末给出运维建议：升级或停用相关服务，并提醒 CISA KEV 的整改期限与近期已观察到的攻击活动。

    Chrome extensions spying on users' browsing data

    作者搭建了在 Docker 中运行 Chromium、经 MITM 代理观测外发流量的自动化扫描框架，用外发流量与访问 URL 长度的相关性指标判定是否泄露浏览历史。扫描结果标记出 287 个疑似外泄的 Chrome 扩展，合计约 3740 万安装量，涉及 Similarweb、Curly Doggo、Offidocs 等多类数据经纪方及关联实体。文中指出浏览历史被收集后可能被转卖用于广告画像、企业内网信息外泄与凭据滥用，并通过蜜罐与端点分析展示了数据被抓取与加密传输的具体迹象。

  7. 07
    Fluorite – A console-grade game engine fully integrated with Flutter

    Fluorite 是一款与 Flutter 深度整合的游戏引擎，可直接用 Dart 编写游戏逻辑并复用 Flutter 的开发工具，同时通过 FluoriteView 在 3D 场景与 UI 组件间共享状态与多视图展示。其核心采用 C++ 实现的数据导向 ECS 架构，兼顾高性能与低端/嵌入式硬件优化，并提供熟悉的高层 API 以便迁移既有开发经验。引擎基于 Google Filament 渲染器和 Vulkan 等现代图形 API，实现接近主机级的 3D 渲染，并支持光照、后期与自定义着色器；同时提供 Hot Reload 以加速迭代。另提供在 Blender 中定义可交互触发区域的能力，便于构建空间化 3D UI 与交互。
