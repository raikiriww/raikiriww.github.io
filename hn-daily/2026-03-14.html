  1. Wired headphone sales are exploding
    comments

    文章指出，有线耳机在被手机取消耳机孔后并未消失，反而因音质性价比更高、连接更稳定而重新流行，形成一种从实用到审美的文化回潮。数据称2025年下半年销量大幅反弹，2026年初收入继续增长，受访者也提到对复杂技术与蓝牙不可靠性的反感推动了回归。作者描述了有线耳机在时尚圈与名人中的象征意义，并提到USB或转接头等连接方案让回归成本降低，但仍存在便携性与易丢失的取舍。

  2. Bucketsquatting is finally dead
    comments

    作者回顾了 S3 桶名全局唯一导致的桶名抢注问题，删除旧桶后他人可注册同名桶并可能获取数据或干扰服务，而可预测的命名规则会放大风险。AWS 现推出面向通用客户的账号命名空间格式（<前缀>-<账号ID>-<区域>-an），只有拥有该命名空间的账号能创建同名桶，错误时会返回 InvalidBucketNamespace。AWS 建议默认使用该格式，并可通过组织级 SCP 的 s3:x-amz-bucket-namespace 条件强制新桶使用，但不会自动保护既有桶，需要新建并迁移数据。

  3. Qatar helium shutdown puts chip supply chain on a two-week clock
    comments

    卡塔尔拉斯拉凡综合设施的氦气生产在遭遇无人机袭击后停摆已九天，若停产超过约两周，工业气体分销商可能不得不迁移低温设备并重新验证供应关系，恢复周期可能拉长到数月。韩国芯片产业受冲击最大，2025年其氦气进口中约64.7%来自卡塔尔，且氦气在硅片冷却中缺乏可替代方案。韩国政府已调查对中东高度依赖的14类半导体材料与设备，韩国企业SK hynix称已分散供应并备足库存，而台积电表示目前影响不大但会持续监测；韩国和台湾各占全球约18%的产能。文章还提到类似情形曾在2022年因俄乌战争引发的氦氖短缺中出现，推动韩国加速供应多元化和本土生产。

  6. 1M context is now generally available for Opus 4.6 and Sonnet 4.6

  8. Hammerspoon
    comments

    Hammerspoon 是一款面向 macOS 的强力自动化工具，核心是把系统功能与 Lua 脚本引擎连接起来。它通过一组扩展暴露系统能力，让用户用 Lua 脚本控制操作系统的多方面行为。项目提供手动安装与 Homebrew 安装方式，并建议在 ~/.hammerspoon/init.lua 中编写配置后开始使用，同时给出入门指南和 API 文档等资源。

  9. Mouser: An open source alternative to Logi-Plus mouse software
    comments

    该项目提供名为 Mouser 的本地化开源工具，用于替代 Logitech Options+，为 MX Master 3S 鼠标重映射 6 个可编程按键，并强调无遥测、无云端、无需账号。它支持按应用自动切换配置、22 类内置动作、DPI 与滚轮方向控制、电量监测、自动重连，以及基于 Qt Quick 的界面和托盘常驻。实现上结合 HID++ 与系统钩子拦截事件，并提供 Windows 与 macOS 的安装运行说明及已知限制（暂不支持 Linux，优先推荐蓝牙）。
