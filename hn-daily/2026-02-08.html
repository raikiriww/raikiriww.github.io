    Monty: A minimal, secure Python interpreter written in Rust for use by AI

    Monty 是用 Rust 编写的极简且偏安全的 Python 解释器，面向 AI 代理执行其生成的代码，并强调无需完整容器沙箱的低成本与低延迟。它支持运行一部分 Python 语法，启动耗时可达微秒级，同时通过可控的外部函数调用隔离文件系统、环境变量与网络，并可做类型检查与状态快照以便恢复执行。功能限制也很明确，包括标准库覆盖有限、缺少第三方库支持以及部分类特性尚未实现，定位于让代理安全执行其编写的脚本。

    Show HN: Look Ma, No Linux: Shell, App Installer, Vi, Cc on ESP32-S3 / BreezyBox

    该项目展示如何把 ESP32-S3 变成“即开即用”的微型电脑，提供迷你 shell、终端、编辑器、编译器与应用安装器，目标是以更小的资源获得类 PC 的交互体验。作者将其命名为 BreezyBox，定位为基于 ESP-IDF 的用户态组件而非完整操作系统，核心提供 vterm/vfs 与常用命令，并可一条命令集成到现有工程。仓库给出在特定 Waveshare 开发板上的示例，强调可移植到其他板卡，并鼓励社区提交更多应用、示例固件与使用场景。

    SectorC: A C Compiler in 512 bytes (2023)

    文章介绍 SectorC：一个用 x86-16 汇编写成、可塞进 512 字节引导扇区的 C 编译器，可能是已知最小的 C 编译器。它支持足够实用的 C 子集，包含变量、函数、if/while、常见运算符、指针解引用、内联机器码与注释等，因此能编写真正可运行的程序。文中还给出示例程序（如绘制移动正弦波）与实现思路，强调在极小体积约束下仍能实现可用的编译器功能。

    Show HN: LocalGPT – A local-first AI assistant in Rust with persistent memory

    LocalGPT 是用 Rust 构建的本地优先助手，强调单一可执行文件、无需 Node.js/Docker/Python，数据与记忆都留在本机。它提供持久化记忆与自治任务心跳，支持 CLI、Web UI 和桌面 GUI，并可接入 Anthropic、OpenAI、Ollama 等多种模型提供方。其记忆以 Markdown 文件组织，结合 SQLite FTS5 与 sqlite-vec 实现关键词与语义检索，并给出完整的安装、配置与命令行/HTTP API 使用方式。

  8. 08
    Vouch

    Vouch 是一个以显式背书为核心的社区信任管理系统，要求贡献者先被社区成员背书，也支持对不当行为者进行点名拒绝，以限制其参与项目的范围。系统采用单一的平面文件记录背书与拒绝信息，格式极简且易于用常见工具解析，并可互相读取不同项目的信任列表，形成跨项目的信任网络。项目提供 GitHub Actions 与 CLI（基于 Nushell）集成，支持在 PR、issue、discussion 中自动检查、更新与管理成员状态。
