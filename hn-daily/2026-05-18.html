  1. The Quiet Renovation at Bitwarden

  2. A nicer voltmeter clock
    comments

    作者分享了第二代“电压表时钟”的设计与制作过程，该设备使用模拟面板电压表替代传统表盘来显示时间。外壳采用枫木材质，通过 Rhino3D 建模并利用 CNC 铣床精细加工，侧壁则通过内部切槽工艺实现无缝弯曲。电路部分基于 AVR128DB28 单片机，无需数模转换器，而是利用高频数字脉冲驱动表头，配合自定义刻度贴纸实现了指针的平滑连续运动。

  3. AI is a technology not a product
    comments

    John Gruber 认为 AI 是一种底层技术而非独立产品，反驳了苹果必须推出“杀手级 AI 硬件”的观点。他指出 AI 将像无线网络一样无处不在地融入所有设备，而非通过某种“AI 代理”彻底取代 iPhone 生态。作者强调苹果的核心理念是交付卓越的产品体验而非单纯的技术，并预测手机在 2030 年仍将是处理 AI 任务的最主要载体。

  4. I turned a $80 RK3562 Android tablet into a Debian Linux workstation
    comments

    rkdebian 是一个针对搭载瑞芯微 RK3562 芯片的 Doogee U10 平板开发的构建系统，支持在不解锁引导程序且不改动原机存储的情况下，通过 SD 卡运行完整的 Debian 12 系统。该项目实现了对触摸屏、Wi-Fi、3D 加速及 NPU 的深度适配，支持运行 Firefox 桌面版并能利用 RKLLM 框架执行本地大模型推理。系统预装了 Phosh 移动 UI 并集成了针对性的电源管理与硬件控制服务，为低成本安卓平板提供了向 Linux 工作站转化的完整方案。

  5. Show HN: Semble – Code search for agents that uses 98% fewer tokens than grep
    comments

    Semble 是一款专为 AI 智能体设计的代码搜索库，利用 tree-sitter 分块、Model2Vec 语义嵌入及 BM25 检索技术，在纯 CPU 环境下实现亚秒级索引与极速查询。相比传统的 grep+read 模式，它能节省约 98% 的 Token 消耗，且无需 API 密钥或外部服务。该工具支持 MCP 协议，可无缝集成至 Claude Code、Cursor 等 AI 开发工具，通过自然语言检索精准提供代码片段。

  6. GenCAD
    comments

    GenCAD是一个基于图像条件的CAD生成模型，它不仅能生成3D实体，还能输出完整的参数化CAD命令历史和程序。该模型结合了Transformer自回归编码器、对比学习框架以及潜空间扩散模型，实现了从图像到可编辑CAD序列的精准生成。相比于传统的网格或点云表示，GenCAD保留了工程制造所需的模型精度与可修改性，为自动化设计流程提供了更高效的工具。

  7. Show HN: Files.md – Open-source alternative to Obsidian
    comments

    Files.md 是一款开源且本地优先的 Markdown 应用，作为 Obsidian 的极简替代方案，它让用户通过纯文本文件管理笔记、日记和任务，且数据完全不离端。该应用支持离线浏览器运行及多种同步方式（如自建服务器或云盘），其代码库设计极其精简且对 LLM 友好，旨在让用户通过思考而非复杂插件来沉淀知识。应用预设了 Chat.md 和 journal 等标准结构，并集成了 Telegram 机器人，方便用户快速记录想法并保持系统的简洁与长久可用。

  8. Garry Tan, the CEO of YC, accused me of unethical reporting
    comments

  9. We stopped AI bot spam in our GitHub repo using Git's –author flag
    comments

    Archestra 团队针对 GitHub 仓库中泛滥的 AI 机器人垃圾内容（AI slop）采取了严厉措施，以解决真实贡献者被淹没以及维护工作不堪重负的困境。由于 GitHub 缺乏原生的白名单功能，他们启用“限制既往贡献者”设置，并巧妙利用 Git 的 --author 参数开发了一套自动授权流程。新贡献者需在官网完成 CAPTCHA 验证，随后由 GitHub Action 代表其提交 commit 以触发权限，从而在确保社区质量的同时有效阻断了 AI 自动生成的低质量干扰。
