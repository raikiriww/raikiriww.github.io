Daily Hacker News for 2026-05-16

The 10 highest-rated articles on Hacker News on May 16, 2026 which have not appeared on any previous Hacker News Daily are listed below, with Chinese summaries for quick reading.

  1. A 0-click exploit chain for the Pixel 10
    comments

    Google Project Zero 披露了针对 Pixel 10 的 0-click 漏洞链，通过适配针对 Dolby 库的 CVE-2025-54957 漏洞实现了初步代码执行。其核心提权手段利用了 Pixel 10 新引入的 VPU 驱动缺陷，该驱动的 mmap 处理函数未对物理内存映射范围进行边界检查，导致攻击者能够直接映射并篡改内核镜像。该高危漏洞仅需极少代码即可实现内核任意读写，目前已在 2026 年 2 月的更新中获得修复。

  2. Bun Rust rewrite: \"codebase fails basic miri checks, allows for UB in safe rust\"
    comments

    Bun 的 Rust 重写版本被指出存在严重安全隐患，代码未能通过 Miri 工具的基本检查，在处理 unsafe 逻辑时因构造无效切片导致了悬挂引用和未定义行为（UB）。报告者批评项目当前的实现质量堪忧，称其为“情绪化编程（vibe coding）”，并指出 AI 生成的 Rust 代码难以保证内存安全。该 Issue 呼吁团队聘请专业的 Rust 开发者进行规范化重构，以解决 safe Rust 中可能触发的内存损坏风险。

  3. U.S. DOJ demands Apple and Google unmask over 100k users of car-tinkering app
    comments

    美国司法部（DOJ）向 Apple、Google 等巨头发出传票，要求获取超过 10 万名 EZ Lynk 汽车应用及硬件用户的详细身份信息，以调查其涉嫌违反《清洁空气法案》、销售绕过柴油车排放控制装置的行为。EZ Lynk 及其律师指责此举严重侵犯了用户的第四修正案隐私权，强调该产品具备合法的车辆诊断与性能监控功能。此案的结果可能为政府利用应用商店数据进行大规模监管执法设定重要的法律先例，并引发了关于个人隐私与联邦环境监管权边界的广泛争议。

  4. ABC News has taken all FiveThirtyEight articles offline
    comments

    Nathaniel Rakich 在 X 平台上指出，ABC News 已将数据新闻网站 FiveThirtyEight 的所有文章彻底下线，原链接现均被重定向至 ABC News 政治频道。此举导致数千页的专业分析与知识内容无法访问，被批评为是对海量知识体系的一次无谓抹除。该事件引起了公众对于历史数据和深度报道保存现状的关注。

  5. Bill to block publishers from killing online games advances in California
    comments

    加州“保护我们的游戏法案”（Protect Our Games Act）取得重要进展，拟要求发行商在关闭在线服务器后提供独立运行补丁或予以退款，以维护消费者的长期访问权。支持者SKG运动强调数字产品不应被无端剥夺，而代表发行商利益的娱乐软件协会（ESA）则辩称游戏仅为授权使用而非所有权，且面临版权续约及技术维护等实际困境。目前法案已通过关键委员会审查，正待州议会全面投票及州长签署，这被视为“停止杀掉游戏”运动在立法层面的显著突破。

  6. 'No way to prevent this,' says only package manager where this regularly happens
    comments

    这篇文章以讽刺笔调描述了 npm 生态系统频繁爆发的供应链攻击，指出开发者往往将其视为不可避免的“自然灾害”，而非源于对深层嵌套且未经审核的第三方包的过度依赖。文章对比了 Go 和 Rust 等拥有健全标准库和严格验证机制的语言，批评 npm 默认执行任意安装脚本的安全隐患，并感叹社区在面对重复性安全违规时的无奈与消极应对。

  7. Frontier AI has broken the open CTF format
    comments

    随着 GPT-5.5 Pro 和 Claude 4.5 等尖端 AI 模型的出现，传统开放式 CTF 比赛格式已宣告终结，AI 能够秒杀大部分中高难度的安全挑战。比赛已演变为编排工具与算力资本的“氪金游戏”，积分榜不再反映真实的个人安全技术，导致初学者的成长路径断裂且顶尖战队纷纷退出。尽管组织者试图对抗 AI 自动化，但收效甚微，CTF 正从一种衡量人类能力的艺术形式退化为缺乏公信力的 AI 基准测试平台。

  8. Moving away from Tailwind, and learning to structure my CSS
    comments

    Julia Evans 介绍了将项目从 Tailwind CSS 迁移至语义化 HTML 与原生 CSS 的过程，通过组件化结构、CSS 变量及原生嵌套选择器建立了一套灵活且可维护的样式系统。布局上利用 CSS Grid 的 auto-fit 和区域模板等特性实现响应式设计，有效减少了对媒体查询和重型构建工具的依赖。此次迁移源于对原生 CSS 特性（如 @import 和嵌套）的支持，以及通过提升专业技能来应对“CSS 很难”挑战并尊重该技术价值的理念转变。

  9. Fecal transplants for autism deliver success in clinical trials (2019)
    comments

    亚利桑那州立大学的研究发现，通过粪便微生物移植（MTT）增加肠道菌群多样性，可显著且长期改善自闭症患者的胃肠道问题及行为症状。2019年的长期随访显示，治疗两年后受试者的自闭症症状平均减少45%，且近半数参与者的评估结果降至临床诊断标准以下。该疗法在成人二期临床试验中表现优于对照组，目前研究团队正积极筹备三期试验以争取获得FDA批准。

  10. SANA-WM, a 2.6B open-source world model for 1-minute 720p video
    comments

    SANA-WM 是一个 2.6B 参数的开源世界模型，支持通过单张图像和相机轨迹在单个 GPU 上生成长达一分钟的 720p 可控视频。其核心设计包括结合了 Gated DeltaNet 与 Softmax 的混合线性注意力机制，以及用于精确 6 自由度轨迹遵循的双分支相机控制架构。该模型采用包含 17B 参数精炼器的两阶段生成流程，在视觉质量媲美工业基准的同时实现了 36 倍的吞吐量提升，其蒸馏版本甚至支持在个人显卡 RTX 5090 上运行。
