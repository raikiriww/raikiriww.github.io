Daily Hacker News for 2026-05-04

The 10 highest-rated articles on Hacker News on May 04, 2026 which have not appeared on any previous Hacker News Daily are listed below, with Chinese summaries for quick reading.

  1. OpenAI’s o1 correctly diagnosed 67% of ER patients vs. 50-55% by triage doctors
    comments

    哈佛大学发表在《科学》杂志的研究显示，OpenAI 的 o1 模型在急诊分诊准确率（67%）和长期治疗方案制定评分（89%）上均显著超过人类医生。尽管 AI 在处理文本医疗记录方面展现出超越临床推理基准的能力，但其局限在于无法捕捉患者的视觉体征与情绪状态，目前更适合作为提供“第二意见”的辅助工具。研究预示未来医疗将向医生、患者与 AI 协作的三方模式转变，但专家同时也对责任归属缺失、医生过度依赖风险及系统安全性等问题提出了警示。

  2. BYOMesh – New LoRa mesh radio offers 100x the bandwidth
    comments

    BYOMesh 是一款小巧且功能强大的双频 LoRa 网格无线电开发板，在单板上集成了支持 sub-1GHz ISM 频段的 SX1276 和支持高速 2.4GHz LoRa 的 SX1281 芯片。该设备旨在通过 2.4GHz 技术提供比传统 LoRa 高出 100 倍的带宽，适用于无需 WiFi 或 HaLow 复杂度即可实现长距离、高带宽网格回传的场景。虽然 2.4GHz LoRa 的数据速率仍低于 WiFi HaLow，但其在功耗控制、授权成本及超长距离覆盖方面具有显著优势。

  3. New statue in London, attributed to Banksy, of a suited man, blinded by a flag
    comments

    伦敦滑铁卢广场最近一夜之间出现了一座署名为 Banksy 的新雕像，描绘了一名身穿西装的男子被手中旗帜遮挡视线，正盲目走向基座边缘，艺术家已通过社交媒体视频确认其身份。该雕像与周边爱德华七世及南丁格尔等历史人物碑林并列，引发了关于现代艺术与公共纪念碑的广泛讨论。目前伦敦市长办公室表示希望保留这件罕见的雕塑作品，并已设置安全护栏以保护日益增多的参观人群。

  4. DeepClaude – Claude Code agent loop with DeepSeek V4 Pro
    comments

    DeepClaude 允许用户在保留 Claude Code 终端交互与自主代理功能的同时，将底层模型替换为 DeepSeek V4 Pro 或 OpenRouter 等兼容后端，实现相同体验下最高 17 倍的成本缩减。该工具通过本地代理拦截 API 请求，支持文件读写、Bash 执行及模型实时切换，并利用 DeepSeek 的自动上下文缓存极大降低了复杂代理循环的费用。虽然在视觉输入和并行工具调用上受限，但其通过 proxy 架构实现了对 Claude Code 生态的高度兼容与远程访问支持。

  5. Let's Buy Spirit Air
    comments

    Spirit 2.0是一项旨在收购于2026年5月2日关停的Spirit Airlines资产的草根运动，计划将其重组为由乘客、员工和社区共同拥有的合作社模式航空公司。该提议借鉴绿湾包装工队的模式，核心在于实行“一人一票”的民主投票权，并根据45美元起步的认缴金额按比例分配潜在利润。目前该项目正处于非约束性出资意向征集阶段，力求抢在私募股权基金介入前整合大众资本，打造一家服务于公众利益而非华尔街股东的航空企业。

  6. GameStop makes $55.5B takeover offer for eBay
    comments

    GameStop 向 eBay 发出 555 亿美元的收购要约，首席执行官 Ryan Cohen 计划通过削减 20 亿美元成本将 eBay 打造为亚马逊的竞争对手。Cohen 提议在合并后担任 CEO 且仅根据公司业绩获取报酬，目前市值约 119 亿美元的 GameStop 已获得 TD Securities 200 亿美元的债务融资承诺。尽管该计划旨在利用 GameStop 门店发展直播电商，但分析师质疑这并非强强联手且会使 eBay 背负巨额债务，消息公布后 eBay 股价上涨 5% 而 GameStop 下跌逾 9%。

  7. Trademark violation: Fake Notepad++ for Mac
    comments

    Notepad++ 作者 Don Ho 发布官方声明，警告网站 notepad-plus-plus-mac.org 冒充官方发布 macOS 版本，属于严重的商标侵权行为。该虚假网站通过盗用软件名称及作者个人简历误导用户和媒体，而作者明确表示 Notepad++ 从未发布过 macOS 版本。目前作者已联系对方要求处理并呼吁社区协助澄清，提醒用户警惕此类未经授权的欺诈项目。

  8. Talking to strangers at the gym
    comments

    针对大学毕业后社交圈受限导致的孤独问题，作者在健身房进行了一场为期一个月的实验，通过每天主动接近一名陌生人来克服社交恐惧并寻找朋友。记录显示他与35位不同背景的人士进行了互动，虽然初期面临巨大的心理压力和偶发的冷遇，但最终成功建立了数段包括共进晚餐在内的深度友谊。作者通过这次尝试意识到接纳尴尬是社交的必经之路，并学会了在持续扩展人脉与深度经营现有关系之间寻求平衡。

  9. Removable batteries in smartphones will be mandatory in the EU starting in 2027
    comments

    欧盟规定自2027年2月18日起，新推出的智能手机和平板电脑必须允许用户使用普通工具轻松更换电池，并禁止使用需加热拆卸的粘合剂。制造商还需提供至少五年的平价备用电池供应及必要的免费工具，旨在通过延长设备寿命减少数百万吨电子垃圾并降低消费者开支。此外，法规还引入了记录碳足迹和健康状态的“电池护照”制度，仅对循环寿命极高或特定专业领域的设备保留豁免条件。

  10. GitHub Is Down
    comments

    GitHub 发生了一起涉及多个服务的重大性能下降事故，影响范围涵盖 Git 操作、Issues、Webhooks、Pull Requests、Actions、Packages、Pages 及 Codespaces。事故表现为多项服务出现高延迟和超时，经过约一小时的紧急修复，目前该故障已得到解决并恢复正常。官方团队正在持续监控系统稳定性，并计划在后续发布详细的根因分析报告。
