    Bus stop balancing is fast, cheap, and effective

    文章指出美国公交站点过密是公交慢、可靠性差和成本高的重要原因，且与欧洲更大的站距形成对比。通过“站点平衡”拉大站距，可在几乎无需新增基础设施的情况下提升速度与准点性，并把节省的运营资源用于提高班次或改善站点质量。多地试点显示删减站点能显著节省乘客时间、提高线路速度与可靠性，同时对覆盖范围影响有限。

  2. 02
    Following 35% growth, solar has passed hydro on US grid

    美国能源信息署公布的2025年数据显示，用电需求同比增长约2.8%，而太阳能发电量同比增长35%，首次超过水电。需求上升幅度超过单一能源增量，导致新增供电除可再生外还动用了更多化石能源，煤电因此回升。天然气设备成本上升与出口政策变化推高了气电成本，使煤电在经济性上相对更可行。机构预计2026年将新增大量太阳能、风电和电池储能容量，风光继续提升占比，但煤电上升可能抵消部分减排效果。

    Google API keys weren't secrets, but then Gemini changed the rules

    文章指出谷歌长期强调 API key 不是秘密，但在启用 Gemini API 后，旧有公开键会被默认赋予敏感访问权限，导致可读取文件、缓存内容并产生高额调用费用。作者扫描公开网页发现 2863 个仍可用的键，并举例说明谷歌自身公开页面上的旧键也能访问 Gemini 接口，构成权限回溯升级与不安全默认配置问题。文中给出披露时间线与整改进展，并建议用户检查项目是否启用了 Generative Language API、审计并限制密钥范围、对外泄键进行轮换并使用 TruffleHog 扫描。

    Jimi Hendrix was a systems engineer

    文章回顾1967年Hendrix录制《Purple Haze》时引入Octavia踏板与一整套模拟链路，强调电吉他被当作可操控的波形合成器，并指出录音室声学与放大器回授也是关键环节。作者尝试用工程化视角拆解这条信号链，基于电路原理解释Fuzz Face、Octavia、wah-wah、Uni-Vibe等效果器如何改变波形与音色，并说明Marshall放大器与空间反馈共同塑造可持续的失真与谐波。为替代“天才凭空出现”的叙事，作者给出可复现的仿真模型与参数（如ngspice与脚本），强调Hendrix与工程师协作、通过系统化迭代实现声音突破。

  8. 08
    Anthropic ditches its core safety promise

    Anthropic 表示将其两年前的安全承诺由硬性约束调整为可变动的非强制框架，理由是原有政策可能限制其在快速竞争的 AI 市场中发展，并且“行业竞赛向上”的期待未能实现。新政策删除了“当模型能力超出可控范围应暂停训练”的条款，改为公开目标与定期报告来提升透明度，但公司也承认新框架更灵活而非强承诺。报道同时指出，公司与五角大楼在 AI 红线问题上存在摩擦，包括对 AI 控制武器和大规模监控的担忧，但称此次政策调整与合同争议无直接关联。
