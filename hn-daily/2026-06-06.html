  1. How LLMs work
    comments

    本文系统性地介绍了现代大语言模型（LLM）的底层工作原理，涵盖了从文本分词、语义嵌入到下一词元预测的全流程。文章详细解析了分词（Tokenization）、嵌入矩阵（Embeddings）以及旋转位置编码（RoPE）等核心组件，并重点阐述了注意力机制中 Query、Key 和 Value 向量的交互方式。文中还探讨了确保模型遵循单向生成规则的因果掩码（Causal Masking）技术，揭示了 Transformer 架构如何有效处理并预测复杂的语言模式。

  3. Did Claude increase bugs in rsync?
    comments

    这份数据分析报告针对 2026 年 5 月 rsync 项目因使用 Claude AI 辅助开发而引发的争议，通过统计 36 个历史版本的“严重程度加权漏洞率”进行了实证研究。分析显示，涉及 AI 辅助的 v3.4.2 和 v3.4.3 版本漏洞指标均处于历史正常区间，其平均漏洞严重度分值甚至低于历史均值。置换检验与费舍尔精确检验等统计学手段均未发现 AI 介入导致漏洞增加的证据，有力反驳了社交媒体上缺乏实证支撑的舆论指责。

  4. New method turns ocean water into drinking water, without waste
    comments

    罗切斯特大学研究人员利用飞秒激光蚀刻的超亲水黑金属面板，开发出一种无需化学预处理且高效节能的太阳能热脱盐技术。该方法利用“咖啡环效应”使海盐以固体形式析出而非产生有害卤水，有效解决了设备结壳堵塞难题并实现了自我清洁。此外，该技术还能通过特定的纳米结构从残留物中提取锂等高价值矿物，为解决全球饮用水短缺和关键材料供应链可持续性提供了新方案。

  5. pg_durable: Microsoft open sources in-database durable execution
    comments

    微软开源了基于 PostgreSQL 的扩展 pg_durable，旨在实现数据库内的持久化执行，使长耗时、容错的 SQL 函数能直接在库内运行。该工具通过 SQL 定义工作流并自动记录检查点，确保任务在系统崩溃或重启后可自动恢复，且无需 Redis 或 Temporal 等外部基础设施。pg_durable 采用 Rust 构建，适用于数据流水线、向量嵌入及外部 API 交互等场景，致力于将计算能力与存储紧密结合。

  6. Gemma 4 QAT models: Optimizing compression for mobile and laptop efficiency
    comments

    Google 发布了采用量化感知训练（QAT）优化的 Gemma 4 模型，旨在通过降低内存占用和提升计算效率，使其能在移动端和日常笔记本电脑上顺畅运行。相较于传统后量化技术，QAT 在训练阶段模拟量化过程以最大限度减少精度损失，成功将 E2B 模型的内存需求降至 1GB 以下。该版本涵盖了通用的 Q4_0 格式及针对移动加速器优化的定制架构，目前已适配 Hugging Face、llama.cpp 和 Ollama 等主流开发生态。
