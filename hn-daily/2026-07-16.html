  1. Running Gemma 4 26B at 5 tokens/sec on a 13-year-old Xeon with no GPU
    comments

    本文介绍了在无GPU且仅支持AVX1的13年旧双路Xeon E5-2690 v2服务器上，成功运行谷歌260亿参数Gemma 4 MoE大模型的实践过程。作者利用Claude诊断并修复了推理引擎ik_llama.cpp在非AVX2硬件上的编译与计算路径缺失问题，通过重写标量循环和调整图构建器实现回退。该系统最终达到了约5.2 tokens/秒的解码速度，相关修复补丁已提交至开源仓库以方便老旧硬件用户。

  2. Grok Build is open source
    comments

    Grok Build 是 SpaceXAI 开发并基于 Apache 2.0 协议开源的终端 AI 编码代理及 TUI 工具。它支持代码库理解、文件编辑、命令行执行、网络搜索和长任务管理，并提供交互式、无头和 ACP 嵌入等多种运行模式。该项目采用 Rust 语言编写，仓库包含了 grok 命令行/TUI 及其代理运行时的核心源代码。

  3. SQLite should have (Rust-style) editions
    comments

    SQLite作为主流本地数据存储库，因向后兼容限制而保留了默认忽略外键约束、宽松的动态类型校验、并发写入冲突时立即报错以及未启用WAL性能优化等四个糟糕配置。虽然用户可以通过配置PRAGMA参数或使用strict表等手段手动修复，但缺乏全局配置机制导致这些缺陷极易被忽略。为此，作者建议SQLite借鉴Rust的“版本”（editions）机制，以便在不破坏旧软件兼容性的前提下更新并改进其默认行为。

  4. The lost joy of music piracy
    comments

  6. How Our Rust-to-Zig Rewrite Is Going
    comments

    Roc语言编译器开发团队在过去一年半中，成功将30万行Rust代码用Zig进行了重写，并实现了与原编译器同等的功能。此次重写旨在解决原有架构下lambda set特化等优化导致的复杂Bug，并应对Rust构建缓慢和全局内存分配器限制等痛点。使用Zig重写不仅为新编译器带来了热代码加载和跨平台编译等特性，还借助其细粒度内存分配控制和对内存不安全代码的辅助机制，提升了开发效率与系统性能。

  7. Sony deletes more movies from the accounts of people who ‘bought’ them
    comments

    索尼 PlayStation Store 因与 StudioCanal 的许可协议终止，计划于2026年9月1日从用户账户中直接删除551部用户已付费“购买”的电影和电视节目，且不提供任何退款或补偿。这已是索尼近年来第三次因版权变动单方面清除用户资产，暴露出数字时代下消费者看似“购买”实则仅获得随时可被收回的临时许可的困境。作者指出，索尼利用复杂的条款刻意隐瞒了这一事实以谋取利益，呼吁监管机构及公众对此进行干预。

  8. Kimi K3: Open Frontier Intelligence
    comments

    月之暗面推出首个开源3T级多模态模型 Kimi K3，参数量达2.8万亿，支持100万 token 上下文，并基于 Kimi Delta Attention 和 Attention Residuals 架构实现了 2.5 倍的缩放效率提升。该模型在长程代码编写、知识工作和推理中表现出前沿性能，成功完成了 GPU 内核优化、MiniTriton 编译器自主开发、芯片设计及自动化科研工作。Kimi K3 目前已在 Kimi 平台及 API 上线，默认启用最大思考模式，并计划于2026年7月27日前开源完整模型权重。

  9. Microsoft Comic Chat is now open source
    comments

    微软宣布将经典的 1990 年代 IRC 聊天客户端 Microsoft Comic Chat 在 GitHub 上开源，该客户端曾协助将 Comic Sans 字体推广给世界。该软件由 David 'DJ' Kurlander 等人于 1995 年开发，能够智能解析聊天文本并自动生成包含独特漫画形象、表情和气泡的漫画分镜。此次开源旨在保存早期互联网的创新历史，代码库中除原始快照外还附带了适配现代 Visual Studio 编译及现代 Windows 运行的现代化改进示例。

  10. Decoy Font
    comments

    Decoy Font 是一种基于空间频率技术设计的防 AI 抓取免费 TTF 字体，在同一空间内融合了前景色干扰字和背景色隐藏信息，人类从远处或眯眼时可看清真实内容。由于 ChatGPT 和 Gemini 等大模型倾向于关注清晰勾勒的前景色，因此在识别该字体的截图时会失败，从而能有效防范 AI 刮擦和 OCR 识别。相比需要动态效果的防 AI 字体，它作为可直接下载安装的 TTF 文件更具易用性，未来可应用于验证码及私密通信等场景。
