Daily Hacker News for 2026-05-15

The 10 highest-rated articles on Hacker News on May 15, 2026 which have not appeared on any previous Hacker News Daily are listed below, with Chinese summaries for quick reading.

  1. New Nginx Exploit
    comments

    DepthFirst 披露了 Nginx 关键漏洞 CVE-2026-42945 的 RCE 概念验证工具 Nginx-Rift，该漏洞源于 ngx_http_rewrite_module 自 2008 年起存在的堆缓冲区溢出问题。攻击者可利用 Nginx 在处理 rewrite 和 set 指令时两次遍历逻辑不一致导致的内存溢出，通过跨请求堆布局篡改内存指针，实现未经身份验证的远程代码执行。此漏洞影响 Nginx 0.6.27 至 1.30.0 之间的广泛版本，建议用户尽快升级至 1.31.0 或 1.30.1 以修复安全隐患。

  2. First public macOS kernel memory corruption exploit on Apple M5
    comments

    Calif 与 Mythos Preview 展示了首个针对 Apple M5 芯片的 macOS 内核内存破坏利用程序，成功绕过了苹果最新的硬件安全防御机制 MIE。该攻击利用数据驱动的内核本地提权链，在 AI 模型辅助下仅耗时五天便完成了开发。研究团队已向苹果提交了 55 页的技术报告，并将在漏洞修复后公布详细的技术细节。

  3. Codex is now in the ChatGPT mobile app
    comments

    OpenAI 将 Codex 集成至 ChatGPT 移动应用，支持用户通过手机实时监控、引导和批准在远程环境运行的编码任务，并同步显示屏幕截图、终端输出及代码对比。此次更新还包括远程 SSH 的正式发布、用于自动化流水线的编程访问令牌以及增强安全合规的 Hooks 功能，同时为企业用户提供 HIPAA 合规支持。该移动端体验已在 iOS 和 Android 预览版上线，覆盖从免费到企业级的所有订阅方案，旨在帮助开发者利用碎片时间高效处理 Bug 修复和代码重构。

  4. New arXiv policy: 1-year ban for hallucinated references
    comments

    arXiv 发布新规，若论文中出现虚假的“幻觉引用”参考文献，相关作者将面临为期一年的禁投处罚。arXiv 计算机科学板块主席强调，作者必须对论文内容承担全部责任，无论内容是由人工还是 AI 生成，旨在维护学术出版的严谨性。

  5. A few words on DS4
    comments

    Redis 创始人 Antirez 介绍了他在一周内开发的本地 AI 项目 DwarfStar 4 (DS4)，该项目利用 DeepSeek v4 Flash 模型和 2/8 bit 非对称量化技术，在高端 Mac 或硬件上实现了可媲美云端旗舰模型的流畅本地推理体验。作者强调 DS4 的成功得益于本地 AI 运动的积累与高效的模型集成，并认为 AI 过于关键而不应仅作为受控服务提供。未来该项目将致力于质量基准测试、集成编码代理以及开发分布式串行与并行推理技术，以保持在高性能本地推理领域的竞争力。

  6. UK government replaces Palantir software with internally-built refugee system
    comments

    英国政府已开发出自主 IT 系统，取代了 Palantir 为“Homes for Ukraine”计划提供的难民安置管理软件，此举成功节省了数百万英镑的开支。该计划最初由 Palantir 免费提供为期六个月的技术支持以确保快速响应，但随着后续费用增加，政府部门决定转向更灵活且具备高安全标准的内部系统。

  7. Mullvad exit IPs are surprisingly identifying
    comments

    Mullvad VPN 并非随机分配出口 IP，而是根据用户的 WireGuard 公钥确定性地在不同服务器的 IP 池中选择相同百分位位置的 IP。这种机制导致用户在切换服务器时仍具有可识别的 IP 指纹，攻击者可通过对比不同服务器的 IP 记录并推算其对应的浮点数范围，以极高准确率实现跨账号关联或去匿名化。为规避此类指纹追踪风险，用户需定期或强制轮换其 WireGuard 公钥。

  8. Explore Wikipedia Like a Windows XP Desktop
    comments

    该项目将维基百科模拟为 Windows XP 桌面环境，允许用户像操作文件系统一样通过文件夹和文档形式浏览百科分类与词条。除了复刻经典的 XP 视觉风格和交互逻辑，工具还集成了维基共享资源（Wikimedia Commons）图片浏览器及一个地理文件探索实验项目，为用户提供复古且新颖的知识探索体验。

  9. Project Gutenberg – keeps getting better
    comments

    古腾堡计划是一个提供超过7.5万本免费电子书的公益图书馆，主要收录美国版权已过期的经典文学作品。平台支持EPUB和Kindle等格式，用户无需注册或安装特定应用即可通过浏览器直接阅读。该项目自1971年起由志愿者协作运行，通过持续的数字化和校对工作，致力于向全球公众免费传播人类文化遗产。
