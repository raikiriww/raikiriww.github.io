    Warcraft III Peon Voice Notifications for Claude Code

    这是一个为 AI 编码代理提供语音提醒的工具，使用《魔兽争霸3》等游戏角色台词，解决代理完成任务或需要权限时没人提醒的问题。它支持 Claude Code、Codex、Cursor、OpenCode 等环境，通过安装脚本注册钩子并下载语音包，可在任务完成、需要输入、会话开始等事件触发提示，并同时更新终端标签和桌面通知。工具实现了 CESP 事件音效规范，提供多平台安装与多种语音包管理、暂停/恢复和配置选项。

    Improving 15 LLMs at Coding in One Afternoon. Only the Harness Changed

    作者强调编码能力差异不只来自模型，关键瓶颈在工具链中的编辑接口与执行“harness”，错误多发生在“模型知道改什么”和“实际写入成功”之间。基准对比了 16 个模型、三种编辑格式，Hashline 在多数模型上胜过 Patch 与 Replace，并显著降低输出 token；例如多模型成功率大幅提升，弱模型收益更大。文章提出用短哈希标记行来定位修改，可避免模型复述旧文本导致失败，并呼吁开放式 harness 优化比单一厂商封闭方案更能提升可靠性。
