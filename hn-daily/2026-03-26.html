  2. Why so many control rooms were seafoam green (2025)
    comments

    本文探讨了20世纪工业控制室（如曼哈顿计划的Oak Ridge和Hanford场址）广泛采用海泡绿背后的设计逻辑。色彩理论家Faber Birren与杜邦公司合作，为工业环境制定了一套功能性色彩安全规范，通过特定的“浅绿色”墙面减轻视觉疲劳、提高效率并减少高压环境下的事故。这套包含消防红、警示橙等标准色的体系在1948年后成为国际通用准则，确保了工业空间在非装饰性的前提下优化了工作效率与安全。

  4. Personal Encyclopedias

  5. Running Tesla Model 3's computer on my desk using parts from crashed cars
    comments
