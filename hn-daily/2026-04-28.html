Daily Hacker News for 2026-04-28

The 10 highest-rated articles on Hacker News on April 28, 2026 which have not appeared on any previous Hacker News Daily are listed below, with Chinese summaries for quick reading.

  1. ASML became the chokepoint for cutting-edge chips
    comments

    ASML 是全球唯一生产极紫外光刻（EUV）机的企业，其设备已成为制造 3 纳米等尖端芯片的战略核心。这些机器通过高能等离子体产生 13.5 纳米波长光，并利用精密反射镜组将复杂电路图案印刷至硅片，其复杂程度堪称世界之最。ASML 的统治地位源于其独特的模块化外包策略、对未成熟技术的长期豪赌，以及在关键时期通过美国主导的 EUV LLC 合作项目获得的科研先机。

  2. Quarkdown – Markdown with Superpowers
    comments

    Quarkdown 是一个现代开源且基于 Markdown 的排版系统，旨在高效创建学术论文、演示文稿、知识库及静态网站。它集成了分页、笔记、文档和幻灯片等多种排版模式，可作为 LaTeX、Notion 及 Docusaurus 等工具的替代方案。该系统具备极速的实时预览功能和图灵完备的脚本能力，支持通过强大的脚本扩展实现复杂排版需求的自动化处理。

  3. China blocks Meta's acquisition of AI startup Manus
    comments

    中国国家发展和改革委员会要求 Meta 撤销对新加坡 AI 初创公司 Manus 价值 20 亿美元的收购，理由是该交易违反了外商投资及出口管制等相关法规。Manus 是一家由中国团队创立并迁往新加坡的通用 AI 代理开发商，因其极快的商业增长被誉为“下一个 DeepSeek”。此次干预体现了北京正加强对本土 AI 技术外流的管控，并对试图通过“新加坡洗白”（Singapore-washing）模式规避中美监管的科技初创企业及风投机构敲响了警钟。

  4. Show HN: OSS Agent I built topped the TerminalBench on Gemini-3-flash-preview
    comments

    Dirac 是一款专注于高代币效率和上下文筛选的开源 AI 编程助手（基于 Cline 分叉），在 Terminal-Bench-2 榜单上使用 Gemini-3-flash-preview 取得了 65.2% 的评分并位列榜首，超越了官方基准。该工具通过哈希锚定编辑、AST 结构操纵和多文件批处理等核心优化技术，在提升代码质量的同时将 API 成本平均降低了 64.8%。它支持自动化工具调用并提供 VS Code 扩展及 CLI，旨在通过精准的上下文控制解决模型在长上下文下的推理退化问题。

  5. Is my blue your blue? (2024)
    comments

    这是一个互动性色彩感知测试，旨在确定用户视觉中“蓝色”与“绿色”的主观分界线。通过让用户对一系列介于蓝色和绿色之间的中间色调进行分类，系统会计算出其感知边界色相值。最终结果会将用户的色彩分类倾向与全球其他参与者的统计数据进行对比，揭示个体间色彩认知的差异。

  6. Talkie: a 13B vintage language model from 1930
    comments

    研究团队推出了 Talkie-1930-13B，这是一个拥有 130 亿参数的“复古”语言模型，完全基于 1931 年以前的 2600 亿个历史文本 token 训练而成，旨在模拟历史对话并研究 AI 在无现代数据背景下的泛化与预测能力。该模型在知识评估上虽逊于现代模型，但在语言理解任务中表现稳健，甚至能通过少量示例学会简单的 Python 编程，展现出跨时代的理解潜力。团队还深入探讨了时间信息泄露、OCR 转录质量等训练挑战，并计划未来将语料库扩展至万亿级，以构建达到 GPT-3.5 能力水平的复古模型。

  7. GTFOBins
    comments

    GTFOBins 是一个专注于 Unix 类可执行文件的精选列表，旨在展示如何利用这些程序绕过配置错误的系统安全限制。它详尽记录了包括 bash、python、curl 在内的数百种工具在获取 Shell、反弹 Shell、任意文件读写及特权提升等方面的具体用法。该资源是安全审计和渗透测试中识别并防范系统配置风险的重要参考指南。

  8. Localsend: An open-source cross-platform alternative to AirDrop
    comments

    LocalSend 是一款免费开源的跨平台文件分享工具，作为 AirDrop 的替代方案，允许用户在局域网内安全传输文件和消息。该应用无需互联网连接或第三方服务器，利用 REST API 和 HTTPS 加密确保通信的安全性与私密性。项目基于 Flutter 和 Rust 构建，全面兼容 Android、iOS、macOS、Windows 及 Linux 等主流操作系统。

  9. Your phone is about to stop being yours
    comments

    Google 计划从 2026 年 9 月起要求全球所有 Android 应用开发者必须进行实名注册并提交政府 ID，否则其应用将被静默拦截，这一政策将终结 Android 长期以来的开放生态。即使是侧载应用也将面临包含 24 小时冷却期在内的 9 步极端复杂流程，实质上赋予了 Google 对所有软件分发的绝对审核权。F-Droid 等开源机构和众多民权组织认为这构成了生存威胁与数字审查，呼吁开发者及用户通过联系监管机构等方式共同抵制。

  10. Ghostty is leaving GitHub
    comments

    Mitchell Hashimoto 宣布 Ghostty 项目将离开 GitHub，主因是其基础设施（如 PR、Issues 和 Actions）频繁的服务中断已严重阻碍专业开发工作。作者虽作为 18 年资深用户对该平台有深厚情感，但指出 GitHub 近期几乎每日的故障使其不再适合进行严肃的软件交付。目前项目团队正讨论新的托管方案并计划分阶段迁移，原 GitHub 地址未来将仅作为只读镜像保留。
