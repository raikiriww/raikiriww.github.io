  1. 01
    Is legal the same as legitimate: AI reimplementation and the erosion of copyleft

    文章以 chardet 由 LGPL 改为 MIT 的 AI 重新实现争议为例，指出“合法”并不等于“正当”，法律只划底线而非背书行为。作者反驳将 GNU 历史类比为正当性依据，强调这次改许可从共益走向私有，削弱了回馈义务与社区信任。文中还批评把 GPL 视为“阻碍分享”的说法，并指出许可选择常与既得利益一致。最后提出当 AI 降低重新实现成本时，更应坚持共益与互惠的规范，而不是以合法性替代正当性判断。

  2. 02
    Florida judge rules red light camera tickets are unconstitutional

    佛罗里达布劳沃德县法官 Steven P. DeLuca 在一份 21 页裁定中撤销一张红灯摄像头罚单，认定相关州法把举证责任从政府转嫁给车主，违反正当程序。裁定指出此类案件虽属民事交通违法，但具有“准刑事”性质，进入县法院后应由政府以“排除合理怀疑”的标准证明司机身份与违法事实。该裁定目前仅适用于布劳沃德县个案，但可能引发其他县及上诉层级的连锁挑战，同时红灯摄像头安全效益的支持方与反对方争议仍在持续。

    OpenAI is walking away from expanding its Stargate data center with Oracle

    文章指出人工智能芯片迭代速度快于数据中心建设周期，导致基础设施投资面临过时风险。报道称OpenAI不再扩展与Oracle在德州Abilene的Stargate数据中心合作，因为希望在别处用更新一代Nvidia GPU形成更大集群，而现有站点预计使用Blackwell且供电一年后才上线。Oracle称相关报道不实但未否认扩建计划，并已在选址、硬件订购和建设上投入巨资；其扩张主要依赖约1000亿美元债务，合作方Blue Owl不再追加资金并计划裁员。文章还提到Nvidia芯片更新已从两年一代加速到一年一代，Vera Rubin性能提升显著，使长期建设的设施更容易落后。

    No, it doesn't cost Anthropic $5k per Claude Code user

    文章质疑“每个 Claude Code Max 用户每月要花 5000 美元算力”的说法，认为这是把零售 API 价格当作真实算力成本导致的误读。作者对比 OpenRouter 上同量级开源权重模型的定价，指出实际推理成本大约只有 API 价格的十分之一，因此最重度用户的真实成本约 500 美元左右，损失规模远小于传闻。文中还解释 5000 美元更像是 Cursor 这类需要支付零售 API 价格的第三方成本，而 Anthropic 的主要亏损来自训练与研发投入，非推理成本。

    I put my whole life into a single database

    作者从2019年起把生活指标集中到单一数据库里，每天跟踪上百类数据，覆盖位置、饮食、运动、睡眠、心情、工作与天气等，并累计约38万条记录。他说明项目目标是用数据回答城市迁移、睡眠与效率、季节与情绪等关系问题，并强调数据可自主管理、可增删问题、可视化完全掌控。页面公开了部分统计与趋势图（以快照呈现保护隐私），同时项目开源并支持自托管。

    Online age-verification tools for child safety are surveilling adults

    美国多州推进针对未成年人的年龄验证法规，导致成年人访问内容也被强制进入验证关卡，引发隐私与言论自由担忧。文章指出平台多依赖第三方身份验证商，用人脸分析、年龄估计或证件核验等方式判定年龄，但数据留存与集中存储带来泄露、执法索取及责任转移风险。Discord等平台的推行计划遭用户反弹并延期，监管方强调需最小化数据使用与安全保护，而隐私倡议者主张应以更强的联邦隐私法取代持续扩张的验证基础设施。
